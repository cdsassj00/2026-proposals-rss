| 1. 지원동기 |
| --- |
| 참여동기 및 기대효과 그동안 생성형 AI를 업무에 부분적으로 활용해 왔지만, 데이터 수집부터 AI 모델 설계·구축·현장 이식까지 전 과정을 체계적으로 다루어 본 경험은 부족한 것이 사 실임. 이번 과정을 통해 단순한 AI 도구 활용 수준을 넘어 실제 행정 현장에 적용 가능한 AI·데이터 분석 역량을 체계적으로 갖추고자 지원하였음 특히 지역경제 위기 예측이라는 실제 업무 과제를 중심으로 아키텍처 설계부터 모델 개발, 현장 이식까지 직접 경험함으로써 현장에서 즉시 활용 가능한 실무 역량을 확보하는 것이 목표 나아가 이 과정에서 습득한 역량을 바탕으로 부내 AX 전환을 촉진하고, 동료 공 무원들이 데이터와 AI를 일상 업무에서 자연스럽게 활용할 수 있는 환경을 만들 어 가는 거점 리더로서의 역할을 충실히 수행하겠음 관련교육 경험 (수강 경험) 데이터 기반 행정과 AI 활용 역량 강화를 위해 꾸준히 관련 교육에 참여해 왔으며, 국가공무원인재개발원, NIA에서 운영하여 인공지능과 데이터 분석 교육에 다수 참여. 특히, 2025년도 대학원 과정을 마쳐 석사 과정을 취득하였고, AI 챔피언 블루 과정도 수료하여 인증을 받았음 (강사 경험) 재난안전교육원의 중견리더과정에서 생성형 AI를 3년째 강의하고 있 으며, 국가기록정보센터, 지자체 등에 부정기적으로 강의를 진행하고 있음. 특히 우리부에서 직원 대상 원데이클래스 강의를 개설하여 생성형AI 및 노코드 솔루션 등의 강의를 진행하고 있음 |

# 2. 실전 AX 과제 기획서

| 1. 과제 개요 |
| --- |
| ○ 과제명 : AI 기반 지역경제 위기 예측 및 모니터링 ○ 추진배경 - 지역경제는 인구 감소, 고령화, 산업 쇠퇴 등 복합적인 요인이 맞물리면서 위기 징후가 점점 빨라지고 있으나, 현재는 생활인구, 인구이동, 인구 구성, 지역경제 지표 등 관련 데이터가 부서별로 분산되어 있어 위기를 사전에 감지하고 선제적 으로 대응하기 어려운 구조 담당자가 매월 데이터를 수기로 수집·가공·분석하는 방식으로는 급변하는 지역 경제 상황에 신속하게 대응하는 데 한계가 있어, AI를 활용하여 다양한 지역경 제 지표를 통합·분석하고 위기를 조기에 예측하는 것이 필요 ○ 목표 생활인구, 인구이동, 인구 구성, 지역경제 지표를 통합한 AI 분석 체계를 구축하 여 지역경제 위기를 사전에 예측하고, 데이터에 근거한 선제적 정책 수립과 의 사결정을 지원 |
| 2. 현안 문제 및 데이터 현황 |
| ○ 문제정의 지역경제 위기를 정확하게 예측하려면 단일 지표가 아닌 다양한 데이터를 복합 적으로 연계하여야 하나 다음과 같은 구조적 문제가 존재 첫째, 생활인구, 인구이동, 고용·소비·사업체 현황 등 지역경제 지표와 같은 핵심 데이터가 부서·기관별로 분산되어 있어 이를 연계·통합하는 데 상당한 시간과 인 력이 소요 둘째, 현재의 분석 방식은 단순 수치 비교나 임계치 설정 수준에 머물러 있어 지역경제 위기의 복합적인 원인을 파악하고 미래 추이를 예측하는 데 한계가 있음 셋째, 매월 수기로 데이터를 취합하고 분석하는 방식으로는 위기 징후를 실시간 으로 감지하기 어렵고, 담당자의 업무 부담도 상당] ○ 보유 데이터 - 생활인구 : 유동인구, 체류인구, 시간대별 인구 분포 - 인구이동 : 전입·전출 현황, 연령대별 인구 유출입 추이 - 인구 구성 : 연령별·성별 인구 구조, 고령화 지수, 생산가능인구 비율 지표(각 기관 공개) : 제조업·소비자 심리지수, 사업체 수, 고용률, 소비 동향, 현황 |
| - 지역경제 지방세·재정 3. 구현 방안 |

| ① 데이터 통합 및 연계 분산된 생활인구, 인구이동, 인구 구성, 지역경제 관련 데이터를 하나의 플랫폼 에서 통합 관리하며, 모든 데이터를 무분별하게 수집하는 것이 아니라 지역경제 위기 예측에 실질적으로 필요한 데이터를 선별하여 연계하고, 한 번 연결된 데 이터는 지속적으로 자동 수집되도록 구성 ② 데이터 자동 가공 및 정제 - 수집된 데이터를 정해진 규칙에 따라 자동으로 표준화하고 정제하여 AI 분석에 최적화된 형태로 변환. 기존에 매월 반복하던 수기 가공 작업을 자동화하여 데 이터 품질을 높이고 담당자의 업무 부담을 경감 ③ AI 기반 지역경제 위기 예측 모델 적용 - 머신러닝·딥러닝 모델을 활용하여 생활인구 감소 추이, 청년층 인구 유출, 소비 위축, 사업체 폐업 증가 등 복합 지표 간의 상관관계를 분석하고 지역경제 위기 를 사전에 예측. 단순 현황 파악을 넘어 위기 발생 가능성과 시점을 예측하는 시계열 분석 모델을 핵심으로 구현. ④ 실시간 모니터링 대시보드 주요 지역경제 지표와 위기 예측 결과를 실시간으로 시각화하는 대시보드를 구 축하여 위기 징후가 감지되면 담당자에게 자동으로 알림을 제공하여 선제적 대응 |

| 4. 현장 이행계획 및 기대효과 |
| ○ 현장 이행계획 1단계 : 특정 시·군 2~3곳을 시범 지역으로 선정하여 AI 위기 예측 모델과 대시 보드를 우선 적용, 2단계: 시범 운영 과정에서 도출된 현장 의견(지표 추가, 알림 기준 조정 등)을 반영하여 모델 및 대시보드 개선, 데이터 자동 수집·정제 파이 프라인의 안정성 점검 및 오류 보완, 3단계: 전체 시·도 대상으로 시스템을 확대 적용하고, 월별 지역경제 분석 업무에 AI 예측 결과를 공식 활용, 4단계: 타 부서 (복지, 일자리 등)의 관련 데이터를 추가 연계하여 분석 범위 확대 분기별 모델 성능 평가 및 재학습 체계를 마련하여 예측 정확도 지속 관리 ○ 기대 효과 - (업무 시간 단축) 현재 매월 수기로 진행하던 데이터 수집·가공·분석·보고서 작성 업무가 자동화됨으로써 담당자 1인 기준 월 평균 약 40시간 이상의 반복 업무가 절감 (정책 대응력 향상) 생활인구 감소, 청년층 유출, 소비 위축 등 복합적인 위기 징후를 AI가 실시간으로 포착하여 담당자에게 신속하게 전달함으로써 정책 골든 타임을 확보 가능 (대국민 서비스 품질 향상) 데이터 기반의 정밀한 지역경제 분석을 통해 지역 특성에 맞는 맞춤형 정책을 수립이 가능하여 실질적인 정책 성과 기대 |

# □ AI · 데이터 거점리더 활동계획

| 1. 데이터기반행정 거점 리더 활동 계획 |
| --- |
| ① 부내 AX 전환 촉진 우리부의 인공지능 전환 우수 사례를 다른 기관 또는 다른 부서로 적극 확산하고, 지역경제 위기 예측 플랫폼 구축 과정에서 축적된 데이터 연계, AI 모델 적용, 자 동화 노하우를 토대로 부내 유사 업무를 발굴하며 AX 전환을 단계적으로 지원 함. 각 담당자와 협력하여 반복적인 수기 업무를 AI 기반으로 전환할 수 있는 과 제를 지속적으로 발굴하고, 데이터 기반 의사결정이 부내 전반의 업무 방식으로 자리잡을 수 있도록 이끌어 감. ② 인재 양성 및 문화 확산 팀원과 부내 구성원을 대상으로 생성형 AI 활용법, 데이터 분석 기초, 프롬프트 작성법 등 실무 중심의 사내 교육을 정기적으로 기획·운영. 단순 교육에 그치지 않고 AI 활용 우수 사례를 발굴하여 부내에 공유하는 문화를 정착시키고, 데이터 를 근거로 업무를 기획하고 보고하는 방식이 자연스럽게 확산될 수 있도록 거점 리더로서 앞장섬. 데이터와 AI가 일부 전문가만의 영역이 아닌 모든 구성원의 일 하는 방식으로 자리잡을 수 있도록 부내 인공지능 전환 문화를 이끌어 가겠음. |
| 2. 과정 참여 필요에 대해 자유롭게 기술 |
| 저는 현재 인공지능·데이터 기반 행정팀장으로서 업무의 인공지능 전환을 이끌어 야 하며, 팀장으로서 AI와 데이터를 단순히 이해하는 수준을 넘어, 실제 업무에 적용하고 팀원들에게 전파할 수 있는 실질적인 역량을 갖추는 것이 무엇보다 중 요하다고 생각함. 이번 과정은 그 역량을 체계적으로 쌓을 수 있는 소중한 기회임. 특히 지역경제 위기 예측이라는 실제 업무 과제를 AI 기반으로 구현하는 과정을 직접 경험함으 로써, 현장에서 바로 활용 가능한 실무 역량을 갖추고자 함. 이론적 이해에 머무 르지 않고 아키텍처 설계부터 모델 개발, 현장 이식까지 전 과정을 직접 다루어 봄으로써 우리 부서 실정에 맞는 AI 활용 모델을 만들어 낼 수 있을 것이라 기대 하고 있음. 또한 이 과정을 통해 습득한 지식과 경험을 팀 내에만 머물지 않고 부내 전체로 확산시키는 것이 제 역할이라고 생각함. 지금도 우리팀에서는 월 4~5회 정도의 교육을 진행하고 있음. 앞으로 동료들이 데이터와 AI를 두려워하지 않고 일상 업 무에서 자연스럽게 활용할 수 있는 환경을 만들어 가겠음. 데이터 기반 행정이 일부 전문가만의 영역이 아닌 우리 부서 모든 구성원의 일하는 방식으로 자리잡을 수 있도록, 거점 리더로서의 역할을 충실히 수행하겠음. |
