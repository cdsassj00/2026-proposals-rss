# 2. 실전 AX 과제 기획서

| 1. 과제 개요 |
| --- |
| ○ 과제명: AI 안전비서(국민안전 24 AI 서비스 도입) 추진배경: 메뉴 기반으로 수동적인 정보 제공 방식은 사용자가 서비스에서 제공하는 정보를 종류를 미리 파악해야한다는 한계점 존재 목표: 재난·안전 정보 특성을 고려하여 사용자의 상황과 성향을 분석하여 가장 필요한 정보를 선제적으로 제시하여 국민 체감도 제고 |
| 2. 현안 문제 및 데이터 현황 |
| ○ 문제정의 - 국민안전24는 다양한 재난안전 정보를 보유하고 있음에도 불구하고, 메뉴 기반 구 조에서는 사용자가 필용나 정보를 직접 탐색해야 하는 한계 존재 - 이로 인해 사용자가 사전에 제공 정보에 대한 이해가 없을 경우 접근성이 낮아 지며, 전반적인 서비스 활용도가 저하되는 문제 발생 보유 데이터 - 대피소 운영·위치 정보(자체), 행동요령(자체), 재난 이력정보(생활안전지도) 등 재 난·안전 업무 전반에 관련된 데이터를 직접 생산 및 연계로 보유 중 |
| 3. 구현 방안 |
| ○ (기능정의) AI 안전 비서 - (관심사 추천) 사용자 위치 기반으로 기상 특보, 재난 문자 등 실시간 정보를 분석 하여 가장 적합한 재난·안전 주제를 실시간으로 추천 - (통합 안내) 사용자가 선택한 주제에 대해 행동요령, 피해 이력, 인근 대피소 위 치 등을 정확하고 손쉽게 제공 ![image](/image/placeholder)

# □ AI · 데이터 거점리더 활동계획

| 1. 데이터기반행정 거점 리더 활동 계획 |
| --- |
| ○ AI 인프라 설계 및 공공 서비스 적용 경험을 바탕으로, 데이터 기반 행정 거점 리더로서 조직 내 AI·데이터 활용 체계를 선도적으로 구축하고자 합니다. - 특히 '재난안전데이터 공유 플랫폼'을 중심으로 외부 데이터와의 API 연계를 활성화하여, NDMS 개편 사업과 연계된 데이터 통합·활용 기반을 고도화할 계획입니다. - 이를 통해 부처 간 데이터 공유 및 협력 체계를 강화하고, 상황 인지·대응에 필요한 데이터를 신속하게 활용할 수 있는 환경을 조성하겠습니다. ○ 또한 AI 안전 비서 구축 및 확산 경험을 기반으로, 데이터 분석 및 AI 활용 역량을 조직 전반에 확산시키기 위한 교육·실습 중심의 인재 양성 체계를 마련하고자 합니다. - 나아가 데이터 기반 정책 수립을 위한 표준 프로세스 및 제도 개선을 병행하여, 경험과 직관이 아닌 데이터에 근거한 의사결정 문화를 정착시키겠습니다. - 이를 통해 공공 서비스 전반에 AI와 데이터가 자연스럽게 내재화된 지속 가능한 혁신 기반을 구축할 계획입니다. |
| 2. 과정 참여 필요에 대해 자유롭게 기술 |
| ○ 기존의 AI 프로젝트 수행 및 교육 이수 경험을 통해 실무 적용 역량을 축적해왔으나, 이를 조직 전체로 확산하고 정책 및 제도 수준까지 고도화하기 위해서는 보다 체계적이고 전문적인 교육이 필요하다고 판단하였습니다. - 특히 데이터 기반 행정 리더로서 요구되는 전략적 기획력과 최신 AI·데이터 분석 기술을 통합적으로 학습함으로써, NDMS 개편 및 AI 서비스 확산을 성공적으로 이끌 수 있는 실행력을 확보하고자 합니다. ○ 과정을 통해 이론과 실무를 아우르는 고도화된 역량을 갖추고, 대국민에게 상황에 맞는 정보를 선제적으로 제공하는 지능형 공공 서비스 구현을 주도하는 핵심 인재로 성장하고자 합니다. |
