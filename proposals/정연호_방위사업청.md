# 1. 지원동기  
- **참여동기**  
  육군 대위('17년 임관)로서 방위사업청 획득전문인력으로 '24. 10월 전입 후 무기체계 사업관리 업무를 수행 중입니다. 사업관리에는 법령과 규정 준수가 필수적이며, 이에 따른 행정 업무가 큰 비중을 차지합니다. AI 기반 자동화를 통해 업무 효율성과 의사결정 속도를 향상시킬 수 있다는 점에서 본 교육에 지원하게 되었습니다. 특히 생성형 AI와 '바이브코딩'을 활용하면 프로그래밍 리터러시만으로도 실무에 적용 가능한 서비스 개발이 가능하다고 판단했습니다.  

- **기대효과**  
  AI 전문 개발 지식 없이도 데이터 분석, 모델링, 웹앱 구현 등 실전 역량을 강화할 수 있을 것으로 기대합니다. 이를 통해 기관 내 반복적 행정업무 자동화를 추진하고, 조직 전체의 업무 효율성을 개선하고자 합니다. 또한, 교육을 통해 습득한 지식을 바탕으로 직원 복지 향상에도 기여할 계획입니다.  

- **관련 교육 경험**  
  - **실무 교육**: ChatGPT 및 데이터 분석 자동화 교육 수료, 국방 AI 도입 사례(Palantir, Microsoft 등) 연구  
  - **학위 교육**: 통계학 석사 과정 중 Python/R을 활용한 데이터 전처리, 시각화, 딥러닝 모델 분석 연구 수행  
  - **핵심 역량**: LLM 모델 경량화(모델 압축 및 파인튜닝), 데이터 전처리, RAG 기반 질의응답 시스템 설계

# 3. 실전 AX 과제 기획서  
## 1. 과제 개요  
- **과제명**: 생성형 AI 및 공공데이터 API 기반 방위사업 규정 실시간 조회·행정업무 자동화 서비스 구축  
- **추진배경**:  
  - 방위사업 업무는 「방위사업법」 등 복잡한 규정 준수가 필수적이나, 법령 조회 시 수작업으로 인해 시간 소요(10~15분) 및 접근성 문제 발생  
  - 생성형 AI를 활용한 업무 혁신 필요성 대두  
- **목표**:  
  - 법령 조회 시간 1분 이내 단축, 반복적 행정업무 자동화를 통한 생산성 향상  

## 2. 현안 문제 및 데이터 현황  
- **문제정의**:  
  - 법령 분철화로 인한 조회 효율성 저하, 외부 환경에서의 접근성 한계, 반복적 행정업무 수작업 문제  
- **보유 데이터**:  
  - 국가법령정보 오픈 API, 공공데이터포털 API, 내부 행정업무 비정형 데이터  

## 3. 구현 방안  
1. **1단계**: RAG 기반 실시간 법령 조회 시스템 구축(국가법령정보 API + 생성형 AI)  
2. **2단계**: 생성형 AI 서비스(DAPADA) 고도화(업무 추천 시스템 추가)  
3. **3단계**: 공문서·보고서 작성 자동화 기능 구현(MCP/Skills, 프롬프트 엔지니어링 적용)  
4. **확장**: 웹앱 기반 통합 서비스 개발(PC/모바일 호환)  

## 4. 현장 이행계획 및 기대효과  
- **이행계획**:  
  1. 개인 프로젝트: RAG 시스템 구현  
  2. 팀 프로젝트: 다부처 협업 AI 서비스 확장  
  3. 시범 적용: 방위사업청 일부 부서 테스트  
  4. 확산: 조직 전체 적용 및 내부 시스템 연계  
- **기대효과**:  
  - 법령 조회 시간 단축, 출장 시 실시간 대응력 향상, 공문서 작성 부담 감소, 범정부 AI 행정 모델 발전
