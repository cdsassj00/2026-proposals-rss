# 1. 지원동기  
### 기관 AI 전환 추진  
KCA는 '25년 AI디지털심화팀 신설, GPU 서버 도입 예산 확보 등 기관 차원의 AI 전환을 적극 추진 중입니다. 자체 LLM 기반 도메인지식 서비스(KCA Al-hub)를 업무망에 정식 오픈하여 부서별 도메인지식을 RAG 파이프라인으로 변환, 자연어 활용이 가능한 환경을 구축했습니다.  

### 거점 리더 역할 수행  
- **KCA Al-hub 기획·개발·운영**: 7개 핵심기능 구현  
- **AI OCR 평가자료 자동화**: 20시간 단축  
- **비전공 직원 대상 바이브코딩 교육**: 5회(15시간), 3개 부서 6명  
- **해커톤 우수상(3위) 수상**: 매일경제·앤트로픽 해커톤 본선  
- **AI챔피언 블루(25-2회) 인증**: AI 개발·모델링 기초 역량 공식 인증  

### 참여 동기  
거점 리더에게 요구되는 기술·업무·리더십 역량을 체계적으로 강화하기 위해 본 과정에 참여합니다. 현장 실무 역량을 바탕으로 AI 리터러시, 데이터 모델링, 자동화 워크플로우 등 이론 및 방법론을 보완하여 '26년 신규 GPU 기반 VisionLLM·추론LLM 활용 AI 모델 구축을 준비 중입니다. ICT자격본부와 한국산업인력공단과의 협업을 통해 국가기술자격 응시 자격 자동심사 AI 모델 개발을 추진하며, 우수 AX 기획서 선정 시 PoC 구현 및 현장 이행까지 연계할 계획입니다.

# 2. 경력 및 실적  
### 1. 자체 LLM 기반 도메인지식 서비스(KCA Al-hub) 기획·구축 ('25.8~12월)  
- **추진 배경**: 외부 AI 서비스 활용 시 내부 데이터 유출 우려로 업무망 내 자체 LLM 인프라 구축 필요  
- **성과**:  
  - GPU 서버(RTX 5090) 조달·설치 완료, Ubuntu 서버 OS 및 CUDA 환경 구성  
  - 16개 컬렉션, 116개 문서 파일, 2,502개 벡터DB 구축  
  - 1,000건 질의 테스트 결과: GPT-OSS 20B(정확도 90.3%), EXAONE 4.0 32B(정확도 88.1%)  
  - 6개 업무영역(인사·조직, 재무·보상 등) 서비스 오픈  

### 2. AI 기반 식수예측(Food-Cast) 고도화 ('25.9월)  
- **성과**:  
  - 머신러닝 앙상블 기법(Ridge, RandomForest 등) 적용  
  - RMSE 44% 감소(1.09 → 0.61) 달성  
  - 13개 REST API 엔드포인트 및 대시보드 완성  

### 3. 자체 LLM AI OCR·외부API 활용 평가자료 자동화 ('26.3월)  
- **성과**:  
  - 서류평가 참고자료 작성시간 20시간(1,200분) 단축  
  - 4단계 AI 파이프라인(데이터 추출·통합·검증·결과 생성) 구축  

### 4. 매일경제·앤트로픽 해커톤 교육 ('26.3월)  
- **성과**:  
  - 15시간(5회) 바이브코딩 교육 운영  
  - 민생위기 조기경보 레이더(IRMI-Radar) 90% 사전 개발  
  - 해커톤 본선 우수상(3위) 수상  

### 5. 경영평가 실사대응 AI도구 활용 교육 기획·운영 ('26.4월)  
- **성과**:  
  - 코딩 지식 없는 직원도 20분 내 AI 시스템 구축 가능한 가이드 제작  
  - Claude Desktop, MCP, Skill(hwpx.skill) 등 4가지 핵심 도구 교육

# 3. 실전 AX 과제 기획서  
## 1. 과제 개요  
- **과제명**: 국가기술자격 응시자격 자동심사 AI 플랫폼 구축  
- **추진 배경**:  
  - 연간 12,000명 응시자격 심사에 2,000시간 소요  
  - 증빙서류 양식 불일치, 담당자별 판단 편차, 민원 리스크 상존  
  - 한국산업인력공단과 데이터 연계로 중복 서류 제출 간소화 필요  

## 2. 현안 문제 및 데이터 현황  
- **수기 심사 비효율**: 연간 2,000시간 단순 반복 업무 발생  
- **심사 일관성 부족**: 담당자별 판단 편차로 민원 리스크  
- **협업과제 중복 서류 제출**: 동일 증빙서류 중복 제출 필요  
- **민원상담 비효율**: 연간 수만 건 민원 발생, 상담사 응대 부담  

## 3. 구현 방안  
### 1단계: 데이터 정비 및 GPU 인프라 구축 (1~2개월)  
- H200 GPU 서버 설치, 벡터DB 확장  
- 학습 데이터셋 1,000건+ 구성, 산업인력공단 API 연계 테스트  

### 2단계: AI 파이프라인 개발 (2~3개월)  
- **응시자격 자동심사**: VisionLLM 서류 인식 + 추론LLM 적격 판정  
- **시험문제 교정(윤문)**: 내부LLM 기반 7대 교정 항목 구현  
- **민원상담 AI Agent**: RAG 기반 답변 초안 생성 및 감정 분석  
- **NCS 코드 자동매칭**: 추론LLM 기반 출제기준-NCS 직무 코드 대응  

### 3단계: 파일럿 운영 및 고도화 (1~2개월)  
- 1~2개 종목 대상 실데이터 운영, AI-담당자 판정 일치율 95% 목표  
- 오판 유형 분석 및 종목별 규칙 튜닝
