- 2025년 12월 AI챔피언 블루 인증 취득 후, AI 기술의 공공 업무 적용 가능성을 체감하며 다수의 AI 활용 프로젝트를 기획·개발·운영  
- 국가인공지능전략위원회로부터 공직사회 디지털·AI 전환 모범 사례로 선정, 헤럴드경제·서울신문·전자신문 등 주요 매체 보도 성과 달성  
- "만드는 역량"은 확보했으나, 데이터 분석 및 정책 활용 체계 미비로 인해 본 과정 지원을 통해 **도구 개발자 → 데이터 기반 의사결정 설계자**로 역할 확장 목표

# 2. 경력 및 실적  
### 📌 AI·데이터분석 프로젝트  
1. **코닥(Kordoc)**  
   - 공공기관 문서(HWP/HWPX/PDF)를 AI 활용 가능 형태로 변환·분석  
   - 5개 공공 프로젝트 검증 완료, GitHub 오픈소스 공개  
   - 기술: Python, 문서 파싱, AI 연동  

2. **Korean-law-mcp**  
   - 법제처 API 기반 17만 5천 건 법령 통합 검색 시스템  
   - 헌법·법률·행정규칙·판례 64개 카테고리 구조화 도구 제공  
   - 기술: TypeScript, MCP, 법제처 Open API  

3. **교육시설 AI 자문 (edu-facility-ai)**  
   - OCR 기반 문서 분류·태깅, 2단계 교육과정 분석, 7개 공공데이터 크롤러 구축  
   - 기술: Tauri, Gemini API, Playwright  

4. **법령비교 플랫폼 (Lexdiff)**  
   - 국가법령 ↔ 자치법규 실시간 비교, AI 자연어 검색, 위임 체계 시각화  
   - 기술: Next.js, Gemini RAG  

5. **Anything (Docufinder)**  
   - 로컬 문서(HWP/DOCX/PDF) 하이브리드 검색 엔진 (키워드 + 의미 검색)  
   - 기술: Tauri, ONNX Runtime, SQLite FTS5

# 3. 실전 AX 과제 기획서  
## 1. 과제 개요  
- **과제명**: 기관 내부 문서 AI 검색 및 유사 업무사례 자동 추천 시스템 "Anything" 구축  
- **추진배경**:  
  - 연간 수천 건 내부문서(기안문·보고서) 생산  
  - 전자문서시스템 검색 한계 (제목·키워드 단순 매칭)  
  - 유사 선례 탐색 시 수동 검토 필요, 신규 전입 시 업무 적응 시간 과다  
- **목표**:  
  - (정량) 업무 선례 검색 시간 70% 단축 (30분 → 10분 이내)  
  - (정량) 신규 전입자 적응 기간 40% 단축  
  - (정성) 부서 간 지식 단절 해소, 조직 학습 역량 강화  

## 2. 현안 문제 및 데이터 현황  
- **문제정의**:  
  - 맥락 기반 검색 불가, 문서 포맷 다양성, 업무 노하우 개인화  
- **보유 데이터**:  
  - 연간 5,000~10,000건 내부 전자문서 (HWP/HWPX 중심)  
  - 코닥(Kordoc)으로 모든 문서 포맷 변환 가능  
  - 법제처 Open API 연동 자치법규 데이터  
  - 민원 처리 이력 데이터 (CSV/DB)  

## 3. 구현 방안  
- **기술적 접근**:  
  - 문서 전처리: 코닥 + PaddleOCR (스캔 문서 OCR)  
  - 하이브리드 검색 엔진:  
    - 키워드 검색: SQLite FTS5 + 한국어 형태소 분석  
    - 의미 검색: KoSimCSE 임베딩 + 벡터 유사도 검색 (usearch)  
    - Cross-Encoder 리랭킹으로 정확도 향상  
  - 유사 업무사례 추천: 문서 간 유사도 매트릭스 기반 자동 추천  
  - AI 질의응답: Gemini API 기반 RAG (자연어 질문 → 근거 문서 답변)  
- **제도적 접근**:  
  - 문서 인덱싱 자동화 파이프라인 구축  
  - 보안 등급별 접근 권한 설계  
  - 부서별 파일럿 운영 → 피드백 수렴 → 전 부서 확산  

## 4. 현장 이행계획 및 기대효과  
- **이행계획**:  
  1. **PoC (1~2개월)**: 소속 부서 문서 대상 프로토타입 구축  
  2. **파일럿 (3~4개월)**: 2~3개 부서 확대 적용, 검색 정확도 튜닝  
  3. **확산 (5~6개월)**: 전 부서 적용, 자동 색인 파이프라인 안착  
  4. **고도화 (7개월~)**: 타 지자체 공유 모델 개발, 업무 프로세스 개선 인사이트 도출  
- **기대효과**:  
  - 연간 1,500시간 절감 (300명 기준 1인 5시간/월)  
  - 기안문·보고서 품질 표준화  
  - 인사이동 시 업무 공백 최소화  
  - 보안 환경 타 공공기관 즉시 적용 가능 (오프라인 작동 설계)

# 4. 데이터기반행정 거점 리더 활동 계획  
## 1. 활동 계획  
- **AI 인재양성 및 문화확산**:  
  - 비전공자 대상 AI 교육 및 동호회 운영 (월 2회)  
  - "AI 점심 세미나" 운영 (실습형 교육)  
  - AI챔피언 블루 경험 기반 멘토링 체계 구축  
- **데이터기반 정책 수립**:  
  - Korean-law-mcp 활용 자치법규 비교 분석 프로세스 도입  
  - Anything 운영 데이터로 부서별 업무 패턴 분석 → 프로세스 개선 제안  
- **외부 데이터 연계**:  
  - 공공 API 활용 가이드 제작·공유 (법제처·공공데이터포털·Kakao Local API)  
  - 자체 개발 AI 도구 오픈소스 공개 (코닥, Korean-law-mcp 등)  
  - 타 지자체 AI 담당자 네트워크 구축  

## 2. 과정 참여 필요성  
- **성과**: AI챔피언 블루 과정 이후 23개 프로젝트 개발, 국가인공지능전략위원회 혁신 사례 선정  
- **한계**: "만들기"에 집중된 프로젝트로 데이터 분석 역량 부족  
- **목표**: 체계적인 데이터 분석 방법론 습득 → **데이터 기반 의사결정 설계자**로 성장  
- **비전**: 검증된 도구 인프라 + 분석 역량 → 공공기관 AI 전환 거점 리더 역할 수행
