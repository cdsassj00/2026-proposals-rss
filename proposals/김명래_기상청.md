# 지원서 정리 문서

## 1. 지원동기  
본 과정에 지원한 이유는 재난 대응 및 공공 행정 분야에 AI와 데이터 기반 의사결정 체계를 실제로 구현하기 위함입니다. 현재 재난대응훈련은 단일 시나리오 중심으로 운영되어 기관 간 협업 및 의사결정 과정이 충분히 반영되지 못하고 있으며, 평가 또한 정성적 판단에 의존하는 한계를 가지고 있습니다. 재난대응훈련 평가관으로서 이러한 문제를 현장에서 직접 확인하였고, 이를 개선하기 위해 재난대응훈련 시뮬레이션 모델을 설계·구현하고 있습니다.  

드론 및 센서 기반 데이터 수집, 딥러닝 기반 이미지 분석 모델(YOLO) 적용, 서버 연동 등 데이터 수집부터 분석, 의사결정까지 이어지는 End-to-End 데이터 기반 시스템을 기획·구축한 경험이 있습니다. 또한 한국생산성본부 AI·데이터 분석 교육과정 이수를 통해 생성형 AI 및 데이터 활용 역량을 강화하고 실제 프로젝트에 적용해왔습니다.  

본 교육과정을 통해 기존 시뮬레이션 모델을 AI 기반 의사결정 지원 시스템으로 고도화하고, 지자체 재난대응훈련에 적용하여 훈련의 실효성과 객관성을 향상시키고자 합니다. 특히 공공부문에 적용 가능한 표준 모델 구축과 데이터 기반 행정 전환을 선도하는 실전형 AI 리더로 성장하는 것을 목표로 합니다.

## 2. 경력 및 실적  
### 데이터 및 AI 기술 활용 경험  
- **드론/센서 기반 데이터 수집 시스템** 구축 및 분석을 통한 시설물 상태 평가·관리 방안 도출  
- **딥러닝 기반 이미지 분석 모델(YOLO)** 적용: 객체 탐지 및 상태 분석 수행, 데이터 전처리·라벨링·학습·검증 전 과정 주도하여 분석 정확도 향상  
  - 관련 **특허출원 3건** 진행  
- **드론 영상 데이터 활용** 건물 외벽·옥상·유리 패킹 등 시설물 결함 분석 및 유지보수 의사결정 지원 모델 구축  
  - 재난 예방 및 공공 안전관리 분야 확장 가능  
- **세종국립수목원 프로젝트**: 식생·환경 데이터 분석을 통한 탄소저감 연계 관리 방안 도출, 공공기관의 데이터 기반 정책 수립 지원  
- **사업연속성관리(BCM) 및 재난관리 컨설팅**: BIA, 시나리오 분석, Monte Carlo Simulation 등을 활용한 데이터 기반 의사결정 구조 설계 및 복구목표시간(RTO) 산정 등 정책 전략 수립  
- **한국생산성본부 AI·데이터 분석 교육** 이수 후 생성형 AI 및 데이터 활용 역량 강화, 행정 의사결정 효율성·정확성 향상에 기여  

### 보유 역량  
- 데이터 수집 → 분석 → 모델링 → 의사결정까지 연결되는 **End-to-End 데이터 기반 행정 적용 역량**  
- 공공 분야 AI·데이터 분석 프로젝트 기획·구축 경험

## 3. 실전 AX 과제 기획서  
### 1. 과제 개요  
- **과제명**: AI 기반 지능형 테이블탑 훈련(TTX) 시뮬레이션 및 의사결정 지원 시스템 구축  
- **추진 배경**:  
  - 현재 재난대응훈련은 고정된 시나리오, 진행자 중심 운영, 정성적 평가 등 한계 존재  
  - 데이터 기반 의사결정 분석·평가 체계 부재로 실제 대응 역량 향상 미흡  
- **목표**:  
  - AI·데이터 기반 의사결정 중심 훈련 체계 전환  
  - 상황 생성·의사결정 분석·결과 예측·데이터 기반 평가 통합 시스템 구축  
  - 공공부문 적용 가능한 표준 모델 마련  

- **정량적 목표**:  
  - 훈련 평가 정량화 비율 30% 이상 향상  
  - 훈련 참여자 의사결정 시간 20% 이상 단축  
  - 데이터 기반 평가 적용률 70% 이상 확보  
  - 1개 이상 기관 시범 적용(PoC)  

- **정성적 목표**:  
  - 절차 중심 → 의사결정 중심 훈련 체계 전환  
  - 정성적 평가 → 데이터 기반 평가 체계 구축  
  - 기관 간 협업 및 복합재난 대응 역량 강화  

### 2. 현안 문제 및 데이터 현황  
- **문제정의**:  
  - 의사결정 데이터 미축적, 훈련 결과 정량 분석 부재, 시나리오 중심 훈련의 한계, 기관 간 협업 부족  
- **보유 데이터**:  
  - 재난훈련 시나리오·사례 데이터, 행동조치 매뉴얼(SOP), 재난 유형별 대응 절차 데이터, BCM·BIA 분석 데이터, 복구목표시간(RTO) 데이터, 훈련 평가 결과 데이터  
- **향후 계획**: 의사결정 로그 및 대응 데이터 지속적 축적 → AI 학습 데이터로 활용  

### 3. 구현 방안  
1. **시스템적 접근 (AI·데이터 기반)**:  
   - 생성형 AI 기반 시나리오 생성 (과거 재난 사례·SOP 데이터 활용)  
   - 데이터 기반 의사결정 분석 (참여자 대응 선택 데이터 패턴 학습 → 최적 대응 추천)  
   - 시뮬레이션 기반 결과 예측 (Monte Carlo Simulation·RTO 데이터 활용)  
   - 데이터 기반 평가체계 구축 (대응 시간·의사결정 흐름·협업 수준 분석)  

2. **제도 및 행정 적용 방안**:  
   - 훈련 운영 지침에 AI 기반 평가·의사결정 요소 반영  
   - 부서 간 협업 체계 구축 (재난·정보화·정책 연계)  
   - 데이터 표준화·관리 체계 구축  
   - PoC 기반 단계적 확산  
   - 교육 및 역량 강화 프로그램 운영  

3. **통합 적용 전략**:  
   - 기술 + 제도 + 운영 결합형 시스템 구축  
   - 재난대응훈련을 경험 중심 → 데이터 기반, 절차 중심 → 의사결정 중심으로 전환  

### 4. 현장 이행계획 및 기대효과  
- **이행계획**:  
  - 2026년 재난대응 안전한국훈련 후반기 시범 적용(PoC) → 효과성 검증  
  - PoC 결과 기반 단계적 적용 확대 및 시스템 고도화  
  - 을지연습과 연계하여 충무계획 검증 도구로 활용  
  - 표준 운영모델·활용 가이드 개발 후 지자체·공공기관 확산  

- **기대 효과**:  
  - **정량적**: 평가 정량화 30% 이상 향상, 의사결정 기반 훈련 적용률 70% 이상 확대, 훈련 운영 효율성 20% 이상 향상  
  - **정성적**: 형식적 훈련 → 의사결정 중심 훈련 전환, 데이터 기반 평가로 객관성 확보, 협업 및 복합재난 대응 능력 강화  
  - **정책적**: 데이터 기반 정책 수립 체계 확립, 재난훈련-비상대비훈련 연계 강화, AI 기반 재난 대응 표준 모델 구축  
  - **확산 효과**: 공공부문 전반 적용 가능한 범용 모델로 발전

## 4. 데이터기반행정 거점 리더 활동 계획  
### 1. 데이터기반행정 거점 리더 활동 계획  
① **데이터 기반 재난훈련 및 의사결정 체계 구축**:  
   - AI 기반 재난대응훈련 시뮬레이션 모델 고도화 → 의사결정 중심 훈련 체계 전환  
   - 테이블탑 훈련(TTX)을 AI 기반으로 고도화 (상황 생성·의사결정 분석·결과 예측·데이터 기반 평가 가능)  
   - 재난대응 안전한국훈련·을지연습 시범 적용 후 단계적 확대  

② **데이터 기반 정책 수립 및 제도 반영**:  
   - 훈련 과정에서 축적된 평가 데이터·의사결정 로그 데이터·복구 분석 데이터를 기반으로 정책 수립 체계 구축  
   - 데이터 기반 평가 결과를 정책·제도에 반영하여 객관적 의사결정 구조 정착  

③ **데이터 연계 및 통합 활용 체계 구축**:  
   - 공공데이터·외부 데이터와의 API 연계 → 재난·기상·환경·시설(GIS) 데이터 통합 분석 체계 구축  
   - 드론·IoT 기반 실시간 데이터 연계 → 현장 중심 의사결정 시스템 확장  

④ **조직 내 AI·데이터 활용 역량 강화**:  
   - 실무 중심 교육·워크숍 운영, 훈련 담당자 대상 데이터 기반 의사결정 교육 실시, AI 활용 사례 공유를 통한 조직 문화 정착  

⑤ **협력 기반 확산 체계 구축**:  
   - 지자체·공공기관과 협력하여 모델 공동 활용, 표준 시스템·운영 가이드 마련 후 단계적 확산  
   - 재난훈련 평가관·전문가 네트워크 활용 → 전국 단위 확산 체계 구축  

### 2. 과정 참여 필요성  
- **공공 행정의 AI 전환**을 실제로 구현하기 위한 실전 역량 확보 필요  
- 재난대응훈련 및 공공 안전관리 분야는 경험과 매뉴얼 중심으로 운영되며, 데이터 기반 의사결정 체계로의 전환이 시급  
- 재난대응훈련 평가관 및 BCM 컨설팅 경험을 통해 구조적 한계 확인 → AI 기반 시뮬레이션 모델 설계·구현 중  
- 본 과정을 통해 **생성형 AI 활용 역량 고도화**, **데이터 분석·모델링 기술 확보**, **정책·제도 적용 역량 강화** 필요  
- 궁극적으로 데이터 기반 의사결정 체계를 공공 행정 전반에 정착시키고, AI 기반 행정 혁신 실현 목표  

### 최종 목표  
- AI 및 데이터 기반 의사결정 체계를 공공 행정에 정착시켜 재난 대응 및 안전관리 분야의 실질적인 행정 혁신 구현
