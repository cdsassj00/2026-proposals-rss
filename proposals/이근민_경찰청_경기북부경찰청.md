| 1. 지원동기 |
| --- |
| - 참여동기 및 기대효과 ○ 저는 2024년부터 경기도북부 데이터센터와 협업을 통한 유동인구 자료를 활용, 경기북부청 범죄예방 분야 우수사례로 선정된 이후 AI · 데이터 분석, 파이썬, 머신 러닝 등 학습을 통해 직접 실무에 적용하는 등 적극적인 업무처리를 추진하였 습니다. 특히 ① 25년 3월 경 '자율방범대 AI 모바일 웹앱'을 제작하여 자율방범대에 배포하였고, 이는 자율방범대에게 취약지역을 제공함과 동시에 LLM모델을 활용 하여 순찰 매뉴얼을 제공해주는 형태였습니다. 실제 자율방범대원의 긍정적인 반응과 범죄가 감소하는 등의 효과로 경기북부청 베스트 자율방범대로 선정되는 가시적인 결과가 나타났습니다. ② 또한 범죄예방 전략회의 시 최근 10년간 발생 건수를 기반으로 머신러닝 시계열 분석, DBSCAN 등 활용한 발생 예측을 통해 취약지를 선정하고 매월 범죄예방 전략을 수립하면서 2025년 5대범죄 5.5%(-210건) 감소하는 효과가 나타났습니다. 위 1,② 사례는 실제로 AI와 데이터 분석을 실무에 직접 적용하면서 범죄예방 효과가 두드러지게 나타나는 효과를 체감하여 더욱 더 해당 부분을 접목하고 싶어졌고 이를 더 공부하기 위해 관련 자격증 취득 공부를 시작하였습니다. 25년 11월 경 데이터분석 준전문가(ADsP) 국가공인 자격증을 취득하여 해당 분야에 대한 이론공부를 함께 진행하였고 26년 4. 11. AI-POT 1급 자격증 응시를 앞두고 있습니다. 실제 업무적용과 스스로 공부 하면서 현장에서 더 도움이 되고 안전한 지역사회를 조성하는데 많은 관심이 생겨 데이터 분석 전문 인재 양성과정에 지원하게 되었습니다. - 관련교육 경험 ○ 2025. 4. 21. ~ 6. 20. 경찰청에서 주관하여 국민대학교에서 교육하는 치안AI · 데이터분석 전문가 양성과정을 수료하였습니다. 해당 교육을 통해 AI와 코딩에 대한 전반적인 이해도를 향상시켰으며 과업 주제로 '내부 데이터 기반 로컬 챗봇' 모델 개발을 진행하였고 RAG 기반으로 PDF 및 이미지 파일에서 관련정보를 추출 하고 검색하는 기능을 구현한 경험이 있습니다. ○ 2025. 6. 30. ~ 8. 12. 경찰청에서 주관하여 (주)모자이크테일러에서 교육하는 「AI 데이터 기반 범죄예방 기능 특화 교육 - ArcGIS를 활용한 범죄예방 실무교육」 과정을 수료였으며 112신고 시간대별 순찰 최적지역 선정 자동화 모델을 제작 하였습니다. 이 모델은 범죄 발생장소 및 112신고 데이터를 지오코딩을 통한 좌표값 으로 변환, 공간 머신러닝 알고리즘인 DBSCAN을 적용하여 범죄 발생장소를 군집화 하여 데이터기반 치안활동을 전환하였습니다. 해당 사례를 통해 '25년 경찰청 적극행정 우수공무원 대상자로 선발되는 성과가 있었습니다. |

| 2. AI·데이터분석 활용 경력 및 실적 |
| --- |
| - AI·데이터 분석 프로젝트 경험 ○ (로컬 LLM모델 제작) 경찰청에서 주관하여 국민대학교에서 교육하는 치안AI · 데이터 분석 전문가 양성과정 수료 시 과업 주제로 '내부 데이터 기반 로컬 챗봇' 모델 개발을 진행한 경험이 있습니다. 해당 모델을 구현하기 위해 Ollama, LM Studio를 활용하여 로컬에서 구동되는 양자화 모델(Llama-3-8B-GGUF)을 선정하여 로컬 기반 챗봇을 제작 하였습니다. RAG 기반으로 PDF 및 이미지 파일에서 관련정보를 추출하고 검색하는 기능을 구현한 경험이 있습니다. - AI·데이터 분석 관련 업무수행 경험 ○ (머신러닝 활용사례) 현재 실제 실무에 적용하고 있는 사례로는 최근 3개년 범죄 발생 데이터를 좌표값으로 변환, 관할 SHP파일에 현출한 후 머신러닝 알고리즘인 DBSCAN을 통해 취약지역을 군집으로 선정 해당 중심지를 Median Center 값으로 추출하여 Reverse Geocoding을 통해 범죄가 집중되는 장소 대상 거점순찰, 탄력 순찰 필요 장소로 선정하여 지역경찰과 공유하고 있습니다. ○ (생성형 AI, 모바일 웹 제작사례) 자율방범대 모바일 웹 · 앱을 자체 제작하여 Google AppScript를 활용하여 자율방범대 AI 모바일 웹앱을 제작하였습니다. 각 자율방범대별 취약지역을 Google Sheet에 입력하여 LLM API와 연동 (Gemini-1.5-Flash)하여 자율방범대의 활동 매뉴얼을 제작해주면서 방범활동을 데이터기반으로 더욱 체계적으로 구축하였습니다. ○ (공공데이터 활용사례) 공공데이터 포털(www.data.go.kr) 내에 있는 고양시 CCTV · 보안등 현황, 종량제 봉투 판매소 및 유동인구 데이터 뿐 아니라 통계지리정보 서비스(SGIS)를 활용하여 범죄예방 관리구역 내 방범시설물 부족 장소를 확인하고 1인 여성가구 밀집지역을 안심귀갓길로 지정하는 등 공공데이터를 활용하여 경찰서 에서 운영중인 범죄예방 강화구역, 안심귀갓길을 재정비하는 업무를 추진하였습니다. |
| 3. 데이터 분석·AI활용 계획 |
| - 수행 예정 프로젝트 < 범죄취약지 예측 고도화, 취약지 · 순찰 필요지역 챗봇 제작> ○ 기존 머신러닝을 활용한 범죄취약지 선정 고도화를 위한 다양한 데이터 결합(날씨, 유동인구, CCTV, 보안등, 유흥주점, 성범죄자 등) 위 결합된 데이터로 선정된 취약장소, 취약시간, 순찰 필요지역을 챗봇을 통해 근무자들에게 제공할 수 있는 시스템 구축 - 지역경찰포털 내 순찰 추천구역을 제공해줄 수 있는 챗봇을 제작하여 탄력순찰 장소 선정 및 근무일지 상 순찰 필요지역을 시간별 제공해주는 기능 구현 |

# 2. 실전 AX 과제 기획서

| 1. 과제 개요 |
| --- |
| ○ 과제명: 범죄취약지 예측 고도화, 순찰 필요지역 등 지역경찰 안내 챗봇 제작 ○ 추진배경 - 기존 경찰 순찰장소(거점, 탄력순찰) 선정 시 데이터 기반 순찰활동이 아닌 경험 기반 순찰활동 장소 선정, 순찰활동의 실질적 효과가 떨어지는 문제점 발생 - 지역경찰 포털 내 다양한 자료들이 있으나 찾기 번거로운 부분이 다수, 포털 내 챗봇을 활용하여 자료 제공하는 플랫폼을 통한 업무 효율화 필요 ○ 목표 - 근무일지 작성 전, 탄력순찰 장소 입력 전 지역경찰 포털 내 챗봇을 통해 순찰 필요장소, 취약지역, 취약시간을 제공해주는 챗봇을 도입, 데이터 기반 순찰활동을 통해 범죄발생 및 112신고 도착시간 단축 등 실제 체감 치안안전도 향상 기여 |
| 2. 현안 문제 및 데이터 현황 |
| ○ 문제정의 - (기존) 매일 주 · 야간 지역경찰 근무일지 작성 시 매 시간 순찰차별 중점 순찰구역을 입력하며 매월 탄력순찰 필요장소 선정 시 이전 순찰장소 위주 또는 지역관서 인근을 탄력순찰 장소로 입력하는 문제가 있음, Geo-pros X, Pre-CAS 등을 활용하여 실제 데이터 기반 순찰 필요장소를 입력하여야 하나 시스템이 별도로 구축되어 있어 취약지를 실제로 입력하기에 번거로운 부분이 있는 문제점 존재 ○ 보유 데이터 - (내부) KICS 사건 데이터(KICS), 112신고 데이터(112시스템), 성범죄자 데이터 - (외부) CCTV · 보안등 등 방범시설, 유동인구 데이터, 전국 격자 데이터(국가데이터포털), 유흥주점 데이터(지자체 홈페이지), 날씨(기상청) |
| 3. 구현 방안 |
| ○ 순찰 필요지역 분석: 최근 3개년 5대범죄 발생지 대상 범죄발생이 많은 장소를 좌표화, 머신러닝 DBSCAN 알고리즘을 통한 취약지 자동 생성 후 DB 구축 - 지역경찰 포털 내 챗봇을 통해 DB 내용을 현출하여 거점 · 탄력순찰 장소 선정 - DB내 지역경찰에 필요한 매뉴얼, 지침 등을 탑재하여 현장에 필요한 자료를 함께 제공해주는 형태의 챗봇 개발 |
| 4. 현장 이행계획 및 기대효과 ○ 현장 이행계획 |
| - 최근 개발된 Geopros-X 내 챗봇 형태와 유사하게 지역경찰 포털 내 챗봇을 도입, 팝업형태로 띄워 지역경찰 포털 내 필요한 내용 입력을 통해 순찰장소, 매뉴얼, 지침, 근무일지 내용 등 검색할 수 있는 형태의 시스템 개발 ※ 직원들이 접근하기 쉬워 별도 교육 등 불필요 할 것으로 판단 ○ 기대 효과 - 실제 범죄가 발생한 장소 일대로 순찰활동이 이루어져 실제 주민들이 체감할 수 있는 범죄예방 및 신속출동의 효과가 발생하여 체감치안안전도 향상 가능 - 탄력순찰 노선 선정, 취약지 분석 등 챗봇 형태로 제공 · 안내되어 순찰필요 장소에 대해 별도 분석 불필요, 매뉴얼 등 제공을 통한 업무 감경 효과 기대 |

# □ AI · 데이터 거점리더 활동계획

| 1. 데이터기반행정 거점 리더 활동 계획 |
| --- |
| - 현재 경찰 인재개발원 범죄예방 분야 외래강사로 활동중에 있습니다. 이러한 경험을 살려 현장 경찰관들이 바로 사용할 수 있는 챗봇 매뉴얼 등을 제작 · 배포하고 교육할 수 있는 역량을 갖추고 있다고 생각합니다. - 공공데이터포털의 API를 연계하여 날씨, 유동인구, 방범시설물 데이터를 통해 취약지역을 선정하는데 더 고도화, 향후 타 부처의 행정 데이터와 경찰 치안 데이터를 매핑할 수 있도록 활동할 예정입니다. |
| 2. 과정 참여 필요에 대해 자유롭게 기술 |
| - 이미 독학으로 ADsP 취득 및 LLM 활용 앱을 제작하여 실무에 적용해 보았습니다. 하지만 개별적인 프로젝트 수준을 넘어, 이를 경찰청 전체 시스템(지역경찰 포털 등)과 안정적으로 연동하고 대규모 데이터를 처리하는 업무를 추진해보고 싶었습니다. 단순한 교육 수료에 그치지 않고, 이번 과정을 통해 제안한 '지역경찰 포털 챗봇'을 성공적으로 안착시켜 경찰의 업무 편의를 향상시키고 국민에게 경찰의 활동이 더 체감될 수 있도록 노력하고자 합니다 |
