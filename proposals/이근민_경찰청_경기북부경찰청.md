# 지원서 정리

## 1. 지원동기  
- **참여동기 및 기대효과**  
  2024년부터 경기도북부 데이터센터와 협업해 유동인구 자료를 활용한 범죄예방 분야에서 우수사례로 선정된 바 있습니다. AI·데이터 분석, 파이썬, 머신러닝 학습을 통해 실무에 직접 적용하며 적극적인 업무처리를 추진했습니다.  
  - **주요 실적**  
    ① 2025년 3월 '자율방범대 AI 모바일 웹앱' 제작 및 배포: LLM 모델을 활용해 순찰 매뉴얼 제공, 자율방범대원의 긍정적 반응과 범죄 감소 효과 발생 → 경기북부청 베스트 자율방범대 선정  
    ② 10년간 범죄 발생 데이터 기반 머신러닝 시계열 분석 및 DBSCAN 적용: 취약지 선정 및 매월 범죄예방 전략 수립 → 2025년 5대 범죄 5.5%(-210건) 감소  
  - **학습 및 자격증**  
    - 데이터분석 준전문가(ADsP) 국가공인 자격증 취득(2025년 11월)  
    - AI-POT 1급 자격증 응시 예정(2026년 4월)  
  - **지원 목적**  
    AI·데이터 분석을 통한 범죄예방 효과를 체감하며, 데이터 분석 전문 인재 양성과정에 지원해 지역사회 안전 강화에 기여하고자 합니다.  

- **관련 교육 경험**  
  - 2025.04.21.~06.20.: 경찰청 주관 치안AI·데이터분석 전문가 양성과정 수료(국민대학교)  
    - RAG 기반 '내부 데이터 기반 로컬 챗봇' 모델 개발: PDF·이미지 파일에서 정보 추출 및 검색 기능 구현  
  - 2025.06.30.~08.12.: 경찰청 주관 「AI 데이터 기반 범죄예방 기능 특화 교육」 수료((주)모자이크테일러)  
    - 112신고 시간대별 순찰 최적지역 선정 자동화 모델 제작: DBSCAN 적용 및 공간 머신러닝 활용 → 2025년 경찰청 적극행정 우수공무원 선정

## 2. 경력 및 실적  
- **AI·데이터 분석 프로젝트 경험**  
  - **로컬 LLM 모델 제작**  
    - 치안AI·데이터분석 전문가 양성과정 중 '내부 데이터 기반 로컬 챗봇' 개발  
    - Ollama, LM Studio 활용해 Llama-3-8B-GGUF 모델 기반 로컬 챗봇 구현  
    - RAG 기술로 PDF·이미지 파일 정보 추출 및 검색 기능 개발  

- **AI·데이터 분석 관련 업무 경험**  
  - **머신러닝 활용 사례**  
    - 3개년 범죄 발생 데이터 좌표 변환 후 DBSCAN 적용: 취약지역 군집화 및 Median Center 값 추출 → Reverse Geocoding으로 거점순찰 장소 선정  
  - **생성형 AI·모바일 웹 제작**  
    - Google AppScript로 자율방범대 AI 모바일 웹앱 제작: Google Sheet 입력 데이터 + Gemini-1.5-Flash API 연동 → 방범활동 매뉴얼 자동 생성  
  - **공공데이터 활용 사례**  
    - 공공데이터 포털 및 SGIS 활용해 방범시설물 부족 지역 확인 → 1인 여성가구 밀집지역 안심귀갓길 지정 및 범죄예방 강화구역 재정비

## 3. 실전 AX 과제 기획서  
### 1. 과제 개요  
- **과제명**: 범죄취약지 예측 고도화 및 순찰 필요지역 챗봇 제작  
- **추진배경**  
  - 기존 경험 기반 순찰활동의 비효율성 문제  
  - 지역경찰 포털 자료 접근성 개선 필요  
- **목표**  
  - 챗봇을 통한 순찰 필요장소·취약지역·취약시간 제공 → 데이터 기반 순찰활동 활성화 및 치안안전도 향상  

### 2. 현안 문제 및 데이터 현황  
- **문제정의**  
  - 근무일지 작성 시 순찰구역 입력의 비체계성  
  - Geo-pros X, Pre-CAS 등 시스템 활용의 번거로움  
- **보유 데이터**  
  - **내부**: KICS 사건 데이터, 112신고 데이터, 성범죄자 데이터  
  - **외부**: CCTV·보안등 현황, 유동인구 데이터, 전국 격자 데이터, 유흥주점 데이터, 날씨 데이터  

### 3. 구현 방안  
- **순찰 필요지역 분석**  
  - 3개년 5대범죄 발생지 좌표화 → DBSCAN 알고리즘으로 취약지 자동 생성 및 DB 구축  
  - 지역경찰 포털 챗봇에 DB 연동: 거점·탄력순찰 장소 선정 및 매뉴얼·지침 제공  

### 4. 현장 이행계획 및 기대효과  
- **이행계획**  
  - Geopros-X 챗봇과 유사한 팝업 형태 시스템 개발 → 별도 교육 없이 접근성 향상  
- **기대효과**  
  - 데이터 기반 순찰로 범죄예방 및 신속출동 효과 증대  
  - 챗봇을 통한 순찰장소·매뉴얼 제공으로 업무 효율성 개선

## 4. 데이터기반행정 거점 리더 활동 계획  
- **활동 계획**  
  - 경찰 인재개발원 외래강사 경험을 활용해 챗봇 매뉴얼 제작·보급 및 교육 진행  
  - 공공데이터포털 API 연계로 날씨·유동인구·방범시설물 데이터 기반 취약지역 선정 고도화  
  - 타 부처 행정 데이터와 경찰 치안 데이터 매핑 추진  

- **과정 참여 필요성**  
  - 독학 및 실무 적용 경험을 바탕으로 경찰청 전체 시스템(지역경찰 포털 등)과 대규모 데이터 처리 업무 확장 희망  
  - 제안한 '지역경찰 포털 챗봇'을 성공적으로 안착시켜 경찰 업무 편의성 향상 및 국민 체감 치안 서비스 구현 목표
