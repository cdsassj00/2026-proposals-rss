# 지원서 정리 (AX 과제 기획서)

## 1. 지원동기  
"글로벌 회계/행정 데이터 한계 돌파와 AI기반 가치 창출"  
194개 전 재외공관의 회계, 행정, 인사 업무를 통합·관리하는 정보시스템을 운영 중입니다. 시스템 특성상 각국의 방대한 데이터가 축적되고 있으나, 기존 규칙 기반 검증 방식으로는 다양한 이상 패턴 식별과 데이터 활용성 극대화에 한계가 있습니다. 회계 투명성 확보와 데이터 무결성 검증을 위해 AI 기술 도입이 필수적임을 절감했습니다. 도메인 지식과 시스템 아키텍처에 대한 이해는 있으나, AI 모델링과 고급 데이터 분석 방법론이 부족한 상황입니다. 이번 과정을 통해 AI/데이터 분석 기술을 내재화하여, 데이터 기반 의사결정(Data-driven decision)을 지원하는 거점 리더로 도약하고자 합니다.

## 2. 경력 및 실적  
### 1. 오픈소스 LLM 기반 행정 규정 기술 검증 (PoC)  
- **수행**: GitHub 기반 오픈소스 LLM 환경 구축 및 공개 행정 규정 데이터 임베딩 후 테스트 수행  
- **성과**: 비정형 텍스트 데이터 AI 학습 적용 가능성 검토 및 생성형 AI 기술의 현업 적용 가능성 확인  

### 2. 데이터 분석 기반 통합 리포트 기초 모델 수립 (PoC)  
- **수행**: Tableau 도입으로 예산 데이터 분석 수행  
- **성과**: 수작업 중심의 통계 산출 방식을 데이터 기반 실시간 가시화 체계로 전환  

### 3. 데이터 기반 정책 기획 및 업무 효율화  
- **수행**: 수기 기반 공관별 예산 집행 방식을 리포팅 체계로 전환  
- **성과**: 공관장의 신속한 의사결정 지원 및 불필요한 행정력 낭비 제거

## 3. 실전 AX 과제 기획서  
### 1. 과제 개요  
- **과제명**: 시차 제약 없는 AI 기반 지능형 행정 헬프데스크(Help Desk) 구현  
- **추진배경**:  
  - 글로벌 시차로 인한 실시간 행정 문의 대응 불가  
  - 단순/반복적 질의로 인한 본부 담당자 피로도 증가 및 답변 품질 편차 발생  
- **목표**:  
  - **정량적 목표**: 본부 접수 단순/반복 행정 문의 건수 50% 이상 감축 및 응답 소요시간 단축  
  - **정성적 목표**: 공관 행정 공백 제로화 달성 및 본부-공관 간 정보 비대칭 해소  

### 2. 현안 문제 및 데이터 현황  
- **문제정의**:  
  - 지구 반대편 공관은 본부 야간 시간에 근무하여 급한 문의나 시스템 장애 시 즉각적 확인 불가  
  - 개정되는 법·규정·매뉴얼이 파일 형태로 산재되어 현장 실무자의 접근성 저하  
  - 본부 담당자 출근 후 누적된 공관 문의를 수동으로 처리해야 하는 비효율 발생  
- **보유 데이터**:  
  - 내부 데이터: 길라잡이 등 내부 규정집, 과거 4년간 축적된 Q&A 및 시스템 매뉴얼  
  - 외부 데이터: 국가재정법, 재외공관 회계사무처리규정 등 관련 법령 데이터  

### 3. 구현 방안  
- **AI·데이터 기술 접근 방안**:  
  - RAG 기반 아키텍처 구현: 내부 지침서와 매뉴얼을 벡터 DB로 구성하여 근거 기반 답변 생성  
- **제도 개선 및 부서 협업 방안**:  
  - 부처 공통 AI 인프라 연계: 중복 투자 방지를 위해 부처 차원의 공통 인프라 활용  
  - 지속 학습 거버넌스 수립: 내부규정 개정 시 AI 지식 베이스에 즉각 연계되도록 프로세스 정립  

### 4. 현장 이행계획 및 기대효과  
- **현장 이행계획**:  
  1. **1단계 (기반 조성)**: 내부/외부 데이터 연계  
  2. **2단계 (시범 운영)**: 본부 직원 테스트 및 공관 순차적 오픈 → 답변 정확도 확인 및 미세조정  
  3. **3단계 (전면 확대)**: 전 재외공관 대상 헬프데스크 전면 오픈 및 기능 고도화  
- **기대 효과**:  
  - 업무 시간 단축: 공관 업무담당자 답변 대기시간 제로화 및 본부 담당자 단순 응대 시간 절감  
  - 비용 절감: 부처 공통 AI 인프라 활용으로 개발비용 절감 및 야간 CS 인력 확충 불필요  
  - 행정처리 표준화: 24시간 균일하고 정확한 지침 기반 행정 가이드 제공으로 전 재외공관 행정 처리 표준화 달성

## 4. 데이터기반행정 거점 리더 활동 계획  
### 1. 데이터 기반 내부 정책 수립  
- AI 기반 헬프데스크 답변 정확도 유지를 위해 규정 개정 시 상시 현행화 프로세스 제도화  
- 비정형으로 방치된 데이터를 AI가 원활히 학습할 수 있도록 지원  

### 2. AI 문화 확산 및 조직 내 AI 협력 강화  
- 성공 사례 전파로 실무진의 AI 도입 공감대 형성  
- 주관 부서 주최 실무협의체 참여하여 추가 과제 도출 등 의견 제공  

### 3. 과정 참여 필요성  
- **대규모 정보화 사업의 즉각적인 현장 적용**:  
  공공 예산이 투입되는 교육을 추상적인 실행이 아닌 현장에 즉시 반영 가능한 완벽한 실행 환경을 보유  
- **도메인 지식과 기술을 연결하는 거점 리더**:  
  성공적 AI 도입은 알고리즘 개발이 아닌 현업의 복잡한 비즈니스 로직과 기존 데이터의 융합에 달려 있음. 재외공관 회계행정시스템 전 주기를 담당하여, 양성과정에서 습득한 AI 기술을 현장 데이터에 빠르게 연결할 수 있음.
