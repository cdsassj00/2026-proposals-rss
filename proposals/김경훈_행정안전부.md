## 2. AI·데이터분석 활용 경력 및 실적

- AI·데이터 분석 프로젝트 경험
| 프로젝트명                         |     | 성과                     |
| ----------------------------- | --- | ---------------------- |
| 500m 격자별 소방력 최적화 배치 분석        |     | KISTI 빅데이터 분석 프로젝트 최우수 |
| 뉴노멀 기후재난 위험징후 분석 및 실시간 공유 서비스 |     | 행안부 소방안전 빅데이터 활용 장려상   |
| 전국 최초 농촌 소방 AI 지능형 훈련 시스템     |     | 소방청 직장훈련 공모전 최우수       |
- AI·데이터 분석 관련 업무수행 경험
1. 데이터 분석(머신러닝 등) 업무활용 경험
(1) 500m 격자별 소방력 최적화 배치 분석 (KISTI 최우수)
분석 대상: 경남 전역 500m 격자 단위 소방 수요·공급 데이터
활용 기법: GIS 공간분석, K-means 클러스터링, 최적화 알고리즘
성과: 소방력 공백지역 도출 → 신규 안전센터 입지 선정 근거 제공

(2) 뉴노멀 기후재난 위험징후 분석 및 실시간 공유 서비스 (행안부 장려)
분석 대상: 기상청 데이터 + 과거 재난 이력 데이터
활용 기법: Random Forest 알고리즘
성과: 기후재난 위험징후 사전 감지 → 선제적 대응체계 구축
2. 생성형 AI 업무활용 경험
(1) AI 기반 소방 민원 답변 자동화 시스템 (인사혁신처장 표창, 소방 소속 유일)
활용 기술: LLM 기반 RAG(검색증강생성) 시스템 구축
적용 범위: 소방 민원 유형별 자동 답변 생성
성과: 민원 처리 시간 단축, 답변 일관성 확보
(2) 전국 최초 농촌 소방 AI 지능형 훈련 시스템 (소방청 최우수, 전국 순회교육)
활용 기술: AI 기반 시나리오 자동 생성, 훈련 결과 분석
성과: 농촌 지역 특성 반영 맞춤형 훈련 체계 구축

## 4. 실전 AX 과제 기획서

1. 과제 개요
○ 과제명
AI 기반 의용소방대 통합 지휘·지원 플랫폼 구축
○ 추진배경
의용소방대는 화재 현장 선도착률 높으나, 정보 단절로 역할 수행에 한계
현장 도착 후에야 건물 구조·위험물·거주자 정보 파악 → 초기 대응 지연
소방서 담당자가 수동으로 의소대 연락·역할 배정 → 골든타임 손실
기상·건물·소화전 등 데이터가 분산되어 현장 활용 불가
○ 목표
구분 목표
정량적 의소대 출동 알림 시간 50% 단축 (수동 연락 → 자동 매칭)
정량적 현장 도착 전 초기대응 정보 제공률 100% 달성
정성적 의소대원의 데이터 기반 상황 판단 역량 강화
정성적 소방-의소대 간 실시간 정보 공유 체계 구축

### 2. 과정 참여 필요에 대해 자유롭게 기술
