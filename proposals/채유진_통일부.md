# 지원서 정리 (AX 과제 기획서)

## 1. 지원동기  
### 참여동기 및 기대효과  
현재 통일부 운영 중인 '북한정보포털'은 방대한 데이터를 보유했으나 단순 자료 나열식 검색에 머물러 있다는 한계가 있습니다. 이에 문제의식을 느끼고, 대학원에서 연마한 **PyTorch 기반 딥러닝(DNN, CNN) 역량**을 실무에 적용해 **'AI 웹 브라우징 기반 지능형 포털'**로의 AX(AI 전환)를 실현하고자 지원합니다.  
- 목표: 검색 중심 UX를 **'대화형 맥락 분석'**으로 혁신  
- 기대효과: 공공 부문 AI 도입 선도 모델 제시  

### 관련교육 경험  
- **AI 분야 대학원생**으로서 Python/PyTorch를 활용한 딥러닝 아키텍처 구현 경험  
- DNN(Deep Neural Network)을 통한 정형 데이터 분류 모델 설계 및 성능 최적화  
- CNN(Convolutional Neural Network)을 활용한 이미지 데이터 특징 추출 및 객체 분류

## 3. 실전 AX 과제 기획서  
### 1. 과제 개요  
- **과제명**: AI 웹 브라우징 기반 북한정보포털 사용자 중심 고도화  
- **추진배경**:  
  - 방대한 북한 정보 DB 보유 but 공급자 위주 정보 나열로 진입장벽 높음  
  - 북한 정보의 특수성(생소한 용어) 및 파편화된 카테고리(정치/경제/사회) 문제  
- **목표**:  
  - 사용자 질문 의도 분석 → 포털 DB + 웹 정보 실시간 융합  
  - 비전문가도 이용 가능한 **디지털 해설사** 구현  

### 2. 현안 문제 및 데이터 현황  
- **문제정의**:  
  - 정보 파편화: 인물/용어/통계 데이터가 산재 → 종합 분석 리포트 생성 어려움  
  - 데이터 시차: 상용 LLM은 공공 데이터 보안·신뢰성(Hallucination) 문제 해결 불가  
- **보유 데이터**:  
  - 정형 데이터: 북한 인물 정보, 지명·통계 데이터, 연도별 주요 일지  
  - 비정형 데이터: 동향 리포트, 학술지, 뉴스 기사, 멀티모달 데이터(이미지/영상)  

### 3. 구현 방안  
- **기술적 아키텍처**: Hybrid RAG(Retrieval-Augmented Generation) 시스템 구축  
  1. **내부 지식 학습**: DNN 최적화 기술로 내부 데이터 벡터화 및 색인 고도화  
  2. **멀티모달 분석**: CNN으로 이미지/영상 데이터 특징 추출 → 텍스트와 연계  
  3. **웹 브라우징 에이전트**: 외부 화이트리스트 웹사이트 실시간 브라우징 → 최신 동향 수집 및 신뢰성 검증  
- **사용자 인터페이스**: 자연어 질의 기능 및 요약 브리핑 UI 적용  

### 4. 현장 이행계획 및 기대효과  
- **이행계획**:  
  - PoC 단계: 내부 데이터 중심 질의응답 모델 구축  
  - 확산 단계: 웹 브라우징 베타 적용 + 답변 검수 시스템 구축  
  - 안정화 단계: TTS(오디오 브리핑) 및 모바일 최적화 → 대국민 서비스 확대  
- **기대효과**:  
  - 정보 탐색 시간 **80% 이상 단축**  
  - 비전문가도 전문 리포트 획득 가능  
  - 공공 데이터 유기적 연결 → 데이터 가치 재창출  
  - 근거 기반 AI 응답 → 대국민 서비스 공신력 제고

## 4. 데이터기반행정 거점 리더 활동 계획  
### 전문 지식 전파 및 내재화  
- **AI 브라운 백 미팅** 운영: LLM 에이전트 활용법 + DNN/CNN 원리 공유 → 부서 내 **'생성형 AI 행정 업무 가이드'** 전파  
- **기술 자문 및 가이드라인 수립**:  
  - RAG 기술 표준 정립 → 환각(Hallucination) 방지  
  - 데이터 정제 가이드라인 마련  

### 실전 AX 과제 성공적 안착  
- **북한정보포털 지능형 에이전트 PoC 추진**: 개발팀과의 기술적 커뮤니케이션 리더 역할 수행  
- **데이터 거버넌스 강화**: 북한 관련 데이터를 AI 학습 최적화 형태로 구조화 → 지속 가능한 분석 환경 구축  

### 부처 내 AX 확산 리더십  
- **통일부 AI 협의체 참여**:  
  - AI 정책 의사결정에 기술적 자문 제공  
  - 타 부서 데이터 분석 요청 시 분석 설계·모델링 지원  
- **AX 거점 리더 역할**:  
  - 통일부 데이터가 살아있는 정보로 재탄생하도록 선도  

### 과정 참여 필요성  
- **대학원 이론 + 실무 문제의식 결합**:  
  - PyTorch 기반 DNN/CNN 연구 → 북한정보포털 지능형 서비스 구현  
- **공공부문 특화 AI 모델 정립**:  
  - 신뢰성·보안 우선의 **'내부 RAG + 외부 웹 브라우징'** 통일부 맞춤형 AX 모델 구축  
- **AX 거점 리더 사명감**:  
  - 통일부 데이터 기반 행정을 지능형 서비스 시대로 도약시키는 전환점 마련
