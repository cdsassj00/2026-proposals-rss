# [통일부] 채유진 - 자기소개서 및 과제기획서

**성명:** 채유진

**소속:** 통일부

---

| 1. 지원동기 |
| --- |
| ○ 참여동기 및 기대효과 · 현재 통일부에서 운영 중인 '북한정보포털'은 방대한 데이터를 보유하고 있으나, 단순 자료 나열식 검색에 머물러 있다는 한계가 있습니다. 저는 이러한 점에 문제의식을 가지고, 대학원 과정에서 연마한 PyTorch 기반의 딥러닝(DNN, CNN) 역량을 실무에 투사하여, 사용자 질문에 실시간으로 대응하는 'AI 웹 브라우징 기반 지능형 포털'로의 AX(AI 전환)를 실현하고자 지원했습니다. 본 과정을 통해 포털의 검색 중심 UX를 '대화형 맥락 분석'으로 혁신하고, 공공 부문의 성공적인 AI 도입 모델을 제시하고자 합니다. ○ 관련교육 경험 · AI 분야 대학원생으로서 Python과 PyTorch를 활용하여 딥러닝 아키텍처를 직접 구현한 경험이 있습니다. 특히, 데이터의 복잡한 비선형 관계를 학습하는 DNN(Deep Neural Network)과 이미지·시각 데이터의 특징을 추출하는 CNN(Convolutional Neural Network) 모델을 설계하고, 손실함수 최적화 및 하이퍼파라미터 튜닝을 통해 성능을 개선한 실습 경험을 보유하고 있습니다. |
| 2. AI·데이터분석 활용 경력 및 실적 |
| ○ AI·데이터 분석 프로젝트 경험 · 대학원에서 PyTorch 기반 딥러닝 아키텍처 구현 및 최적화 프로젝트를 수행하며, 다층 레이어 구조의 DNN을 구축하여 정형 데이터 분류 모델을 제작하였습니다. 또한, CNN을 활용하여 시각 데이터의 특징 맵(Feature Map)을 생성하고 객체를 분류하는 과제를 수행하며 비정형 데이터 처리 역량을 쌓았습니다. 이를 통해 AI 모델의 내부 동작 원리를 깊이 이해하고 코드로 직접 제어할 수 있는 기술적 기반을 마련했습니다. ○ AI·데이터 분석 관련 업무수행 경험 · 통일부에서 AI·빅데이터 기반으로 구축된 웹사이트인 북한정보포털 담당자로서 북한 관련 인물, 용어, 지형, 동향 등 방대한 공공 데이터의 거버넌스를 관리하고 있습니다. 기존의 키워드 검색 기반 시스템에서 발생하는 정보 탐색의 비효율성을 진단하고, 이를 LLM 및 AI 에이전트 기술로 해결하기 위한 기술적 타당성 검토와 서비스 고도화 기획 업무를 수행하고 있습니다. |

| 3. 데이터 분석·AI활용 계획 |
| --- |
| ○ 수행 예정 프로젝트 'AI 웹 브라우징 기술을 적용한 사용자 중심 북한정보포털 서비스 고도화(AX)'를 추진할 계획입니다. 수행 초기에는 내부 데이터와 DNN 기반의 분석 모델을 연계하여 답변의 신뢰성을 확보하고, 점차적으로 웹 브라우징 기술을 결합하여 실시간 북한 동향까지 통합 브리핑하는 서비스를 구현해 보고자 합니다. ○ 업무 활용계획 · 본 교육 과정에서 도출할 PoC(기술 검증) 결과물을 바탕으로 AI·빅데이터 기반 웹사이트인 북한정보포털 담당자로서 AI 운영 체계를 정립하겠습니다. 대학원에서 배운 기술적 깊이와 이번 교육의 실무 능력을 결합하여, 부처 내 AI 전환 거점 리더로서 동료들에게 AI 에이전트 활용법 및 데이터 전처리 노하우를 전파하고, 통일부 내 데이터 기반 행정 문화를 확산시키겠습니다. |

# 2. 실전 AX 과제 기획서

| 1. 과제 개요 |
| --- |
| ○ 과제명 : AI 웹 브라우징에 기반한 북한정보포털 서비스의 사용자 중심 고도화 ○ 추진배경 · 현재 북한정보포털은 북한 인물 정보, 주간/월간 동향 등 북한 관련 정보에 대한 방대한 DB를 사실 기반의 데이터로 축적하여 제공하고 있음 · 워드클라우드, 연관어 분석 등 빅데이터 시각화 툴도 제공하고 있으나, 현재의 공급자 위주의 정보 나열 방식은 일반 국민에게 높은 진입장벽을 형성함 * 북한 정보의 특수성(생소한 용어 등), 파편화된 카테고리(정치, 경제, 사회 등) 등을 해결하기 위해 사용자의 의도를 파악하는 맞춤형 가이드 필요 ○ 목표 · 사용자 중심 지능형 브라우저 : 사용자의 질문 의도를 분석하여 포털 DB와 웹 정보를 실시간으로 융합 해설 중심 서비스 전환 : 북한에 전문 지식 없이도 누구나 이용 가능하도록 정보 접근의 용이성을 개선한 디지털 해설사 구현 |
| 2. 현안 문제 및 데이터 현황 |
| ○ 문제정의 · 정보 파편화 : 인물, 용어, 통계 데이터가 개별 메뉴에 산재하여 종합적인 분석 리포트 생성이 어려움 데이터 시차 : 기존 시스템은 업데이트 주기에 따라 최신 동향 반영이 늦으며, 상용 LLM은 공공 데이터의 보안 및 신뢰성(Hallucination) 문제 해결을 못함 ○ 보유 데이터 · 정형 데이터 : 북한 인물 정보, 지명 및 통계 데이터, 연도별 북한 주요 일지 등 · 비정형 데이터 : 북한 주간 동향 리포트, 학술지, 뉴스 기사, 멀티모달 데이터 (이미지 및 영상) 등 |
| 3. 구현 방안 |
| ○ 기술적 아키텍처 : Hybrid RAG(Retrieval-Augmented Generation) 시스템 구축 · 1단계(내부 지식 학습) : DNN(Deep Neural Network) 최적화 기술 활용, 내부 데이터의 벡터화 및 색인 성능을 고도화하여 질문에 맞는 정확한 근거 데이터 추출 · 2단계(멀티모달 분석) : CNN(Convolutional Neural Network) 최적화 기술 활용, 포털 내 이미지/영상 데이터의 특징을 자동 추출하고 텍스트와 연계 |

| · 3단계(웹 브라우징 에이전트) : 외부 화이트리스트 웹사이트를 실시간 브라우징 하여 최신 동향을 수집하고, 내부 데이터와 비교·대조하여 답변의 신뢰성 확보 ○ 사용자 중심의 인터페이스로 전환 : 검색창 대신 자연어 질의 가능한 기능 및 요약 브리핑 UI 적용 |
| --- |
| 4. 현장 이행계획 및 기대효과 |
| ○ 현장 이행계획 · PoC 단계 : 내부 데이터(인물, 용어 등) 중심의 질의응답 모델 구축 및 기술 검증 · 확산 단계 : 웹 브라우징 기능을 베타 적용하고, 답변 검수 시스템 구축 · 안정화 단계 : 오디오 브리핑(TTS) 및 모바일 최적화를 통해 전 부처 및 대국민 서비스로 확대 ○ 기대 효과 · 사용자 편의성 혁신 : 정보 탐색 시간을 평균 80% 이상 단축하고, 비전문가도 전문적인 북한 분석 리포트 획득 가능 · 데이터 기반 행정 실현 : 파편화된 공공 데이터를 AI가 유기적으로 연결하여, 데이터의 가치를 재창출하고 공공부문 AX의 선도적 모델 제시 신뢰성 확보 : 근거(출처)가 명확한 AI 응답을 통해 생성형 AI의 한계를 극복하고 대국민 서비스의 공신력 제고 |

# □ AI · 데이터 거점리더 활동계획

| 1. 데이터기반행정 거점 리더 활동 계획 |
| --- |
| ○ 전문 지식 전파 및 내재화 · AI 브라운 백 미팅 운영 : 본 과정에서 습득한 LLM 에이전트 활용법과 대학원 에서 학습한 DNN/CNN의 원리를 결합하여, 부서 내 동료들에게 '생성형 AI의 행정 업무 적용 가이드'를 공유하겠습니다. · 기술 자문 및 가이드라인 수립 : AI 도입 시 발생할 수 있는 환각 현상(Hallucination) 방지를 위한 RAG(검색 증강 생성) 기술 표준을 정립하고, 데이터 정제 가이드 라인을 수립하겠습니다. ○ 실전 AX 과제의 성공적 안착 · 북한정보포털 지능형 에이전트 PoC 추진 : 본 과정에서 기획한 AI 웹 브라우징 기반 북한정보포털 서비스를 실제 시스템에 이식하기 위해 개발팀 등 업무 이 해관계자와의 기술적 커뮤니케이션 리더 역할을 수행하겠습니다. 데이터 거버넌스 강화 : 파편화된 북한 관련 데이터(인물, 지명, 통계 등)를 AI 학습에 최적화된 형태로 구조화하여, 향후 지속 가능한 AI 분석 환경 구축의 토대를 마련하겠습니다. ○ 부처 내 AX 확산 리더십 · 통일부 AI 협의체 참여 : 본 교육을 통해 획득한 AI 챔피언 블랙 등급 역량을 바탕으로 부처 내 AI 관련 정책 의사결정에 기술적 자문을 제공하며, 타 부서의 데이터 분석 요청 시 분석 설계 및 모델링을 지원하는 거점 역할을 수행할 것 입니다. |
| 2. 과정 참여 필요에 대해 자유롭게 기술 |
| [준비된 기술력과 절실한 실무 문제의식의 결합] ○ 대학원 이론을 현업의 해답으로 바꾸기 위한 기회 · 현재 AI 분야 대학원에서 PyTorch를 활용한 DNN, CNN 구현 등 딥러닝의 핵심 기술을 수련하고 있습니다. 이러한 학문적 연구를 넘어, 제가 매일 마주하는 북한정보포털의 방대한 데이터를 어떻게 하면 국민이 체감할 수 있는 지능형 서비스로 바꿀 수 있을지에 대한 고민이 깊습니다. 본 과정은 저의 기술적 기반을 실무적인 기획 역량과 AX 리더십으로 승화시킬 수 있는 유일한 통로라고 생각합니다. |

# ○ 공공부문 특화 AI 모델 정립의 시급성

· 공공기관의 데이터는 신뢰성과 보안이 최우선입니다. 따라서, 북한 정보와 같은
민감하고 전문적인 데이터는 단순한 상용 AI 도입만으로는 한계가 있습니다.

저는 대학원에서 다지는 모델 최적화 역량과 본 과정에서 배울 실전 프로젝트
수행 능력을 결합하여, '신뢰할 수 있는 내부 RAG'와 '확장성 있는 외부 웹 브라
우징'이 공존하는 통일부 맞춤형 AX 모델을 반드시 구현하고 싶습니다.

○ AX 거점 리더로서의 사명감

· 우리 부처가 보유한 소중한 데이터 자산이 자료 나열식 검색에 머물지 않고,
AI를 통해 살아있는 정보로 재탄생할 수 있도록 앞장서겠습니다.

이번 교육 참여는 저 개인의 성장을 넘어, 통일부의 데이터 기반 행정이 지능형
서비스 시대로 도약하는 전환점이 될 것입니다.
