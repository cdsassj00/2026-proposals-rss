# 1. 지원동기  
○ **참여동기 및 기대효과**  
'23년 '석유제품 이상징후 예측 시스템' 및 '25년 '위험도 예측 시스템 고도화' 사업을 성공적으로 이끌어 산업통상부 장관상(정부혁신 우수사례)과 과학기술정보통신부 장관상(AI 소프트웨어 우수발주)을 수상한 성과를 바탕으로, 조직 차원의 AX(인공지능 전환)를 선도하는 'AI 챔피언 블랙 등급' 거점 리더로 도약하고자 지원합니다. 두 차례의 장관상 수상으로 입증된 기획력과 실무 능력을 이번 교육 과정과 결합해 행정안전부가 지향하는 공공부문 AI 혁신의 표준 모델을 정립하겠습니다. 특히 실전 프로젝트를 통해 확보한 PoC(Proof of Concept) 결과물을 실제 서비스로 이행하여 데이터 기반 행정의 가시적 성과를 증명하고자 합니다.  

○ **관련 교육 경험**  
- 행정안전부(data.go.kr) 데이터 분석 및 AI 활용 역량 교육 이수  
- Python, SQL, 머신러닝(분산분석·상관분석·군집분석) 자기주도 학습 및 실무 적용

# 2. 경력 및 실적  
## AI·데이터분석 활용 경력 및 실적  
### 프로젝트 경험  
1. **석유제품 수급보고시스템 고도화 사업** (2023.07~2023.12, 6,200만 원)  
   - **수행 역할**: 위험도 예측 분석 모델 구축 총괄, AI 분석 솔루션(EC-Miner) 도입 기획 및 관리  
   - **성과**:  
     - AI 빅데이터 분석 솔루션 도입으로 불법유통 이상징후 핵심 인자 75개 도출  
     - LDA 기반 위험도 예측모델 구축 및 업체별 위험등급 4단계 분류 체계 마련  
     - 수급보고시스템 UI·성능·데이터 정합성 전면 개선  
     - '석유품질 안전지도' 서비스로 '24년 우수 행정 및 정책사례 선발대회 수상  

2. **위험도 예측 시스템 고도화** (2025.05~2023.08, 1,500만 원)  
   - **수행 역할**: 컨설팅 발주·관리, 현장 검사원 의견 수렴 총괄, 모델 개선 방향 의사결정  
   - **성과**:  
     - ANOVA·상관분석·군집분석 기법 적용으로 기존 38개 인자의 유의성 검증 및 2개 인자 추가(총 40개)  
     - 비선형 보조모델(KNN) 도입으로 최신 적발 트렌드에 실시간 대응하는 이중 모델 체계 구축  

### 업무 경험  
1. **무자료 거래 이상징후 조기 감지 모델 자체 개발** (2025.08~10.)  
   - 관리원 수급보고 데이터와 외부 공공데이터(오피넷 판매가격 API, 유가보조금 지급률 등) 연계 분석 체계 구축  
   - 입·출하량 불일치, 지역 내 가격 랭킹 등을 핵심 인자로 LDA 기반 예측·탐지 모델 개발  
   - 매주 전국 약 700개 의심 업체 자동 선별 → 현장 단속 효율성 향상  

2. **공공데이터(기상청 API) 연계 주유소 품질관리 자동 알림 서비스 운영**  
   - 과거 3년간 수분 혼입 주유소 강수량 데이터 분석 → '12시간 이내 108mm 이상' 위험기상정보 기준 도출  
   - 기상청 단기예보 API와 연동해 위험 지역 주유소에 자동 문자 발송 시스템 구축  
   - 전년 동기 대비 품질부적합(물과 침전물) 발생 33% 감소

# 3. 실전 AX 과제 기획서  
## 1. 과제 개요  
- **과제명**: AI 기반 이동판매차량 객체탐지 시스템 구축  
- **추진배경**: 공공부문 AI 대전환 전략에 맞춰 단순 반복 업무에 AI(비전, 머신러닝) 기술 도입  
  - 현재 이동판매차량 단속 시 검사원이 채증 영상을 수동으로 전수 검토 → 행정력 소모, 피로도 누적, 휴먼 에러 발생  
- **목표**:  
  - (정량) 채증영상 분석소요 시간 99% 이상 단축(24시간 → 10분 이내)  
  - (정량) 1,000장 이상의 현장 데이터 기반 학습 데이터셋 구축 완료  
  - (정성) 인력 의존적 업무 방식을 AI 기반 과학적 단속 시스템으로 전환  

## 2. 현안 문제 및 데이터 현황  
- **문제정의**:  
  - 과도한 행정력 소모, 피로도 누적·휴먼 에러, 불법개조 차량 식별 어려움, 단속 효율성 한계  
- **보유 데이터**:  
  | 데이터 종류 | 주요 내용 | 규모 / 출처 | 활용 여부 |  
  | --- | --- | --- | --- |  
  | 이동판매차량 실물 이미지 | 현장 촬영 이미지(저장탱크·주유기 등) | 약 1,000장 / 검사원 촬영 | 구축 완료 |  
  | 라벨링 데이터셋 | 이동판매차량 주요 형태 및 특징 | 1,000장 / 자체 구축 | 즉시 활용 가능 |  
  | 현장 채증 영상 | 단속 목적 영상(최대 수 일 분량) | 수십 GB / 기관 보유 | 즉시 활용 가능 |  
  | 추가 이미지(수집 예정) | 불법개조 이동판매차량 유형 | 수집 예정 / 현장 확보 | 수집 중·확대 예정 |  

## 3. 구현 방안  
- **AI·데이터 기술 활용 방안**:  
  1. **YOLOv11m 객체탐지 모델**  
     - 영상 내 객체의 종류(Class)와 위치(Location) 동시 파악  
     - 탐지 속도와 정확도(mAP) 균형 확보, 복잡한 환경에서도 실시간 추적 성능 우수  
  2. **구현 환경 및 자동화**:  
     - Python 3.12 + Ultralytics Framework 기반 구축  
     - CNN 구조로 차량 윤곽선→바퀴/탱크→전체 형태 순 계층적 학습  
  3. **현장 적용 프로세스**:  
     | 단계 | 구분 | 내용 |  
     | --- | --- | --- |  
     | 1 | 데이터 구축 | 현장 촬영 이미지 수집 → 이동판매차량 형체 직접 레이블링 → 학습 데이터셋 완성 |  
     | 2 | 모델 학습 | Python 환경에 데이터 입력 → 학습 파라미터 실행 → 모델 최적화 및 결과 저장 |  
     | 3 | 현장 적용 | 채증 영상 입력 → AI 모델 자동 분석 → 이동판매차량 출현 시간대 스틸컷 추출 |  
     | 4 | 결과 활용 | 담당자에게 핵심 구간 제공 → 정밀 분석 및 단속 전략 수립 |  

- **제도·행정 절차 개선**:  
  - '이동판매차량 AI 단속 보조 시스템 활용 지침' 제정  
  - 현장 검사원 대상 교육 실시 및 피드백 수렴을 통한 모델 지속 개선

# 4. 데이터기반행정 거점 리더 활동 계획  
## 1. 데이터기반행정 거점 리더 활동 계획  
- **외부 데이터 연계 API·플랫폼 활용 계획**:  
  - 국세청 매출 데이터 API, 국토부 유가보조금 데이터 통합 분석  
  - 행정안전부 공공데이터 포털(data.go.kr) API를 전국 GIS 정보와 연계  
  - 이동판매차량 AI 탐지 시스템과 기존 시스템 통합 단속 플랫폼 고도화  

- **데이터 분석·AI 활용 인재 양성 및 문화 확산**:  
  - 팀 내 'AI 활용 사례' 월 1회 공유(객체탐지 원리, 생성형 AI 프롬프트 등)  
  - 타 부서 대상 '비전문가도 이해하는 AI 입문 워크숍' 개최  
  - 'AI 활용 사례집' 제작·배포  

- **데이터기반 정책 수립·제도 마련**:  
  - AI 시스템 고도화 로드맵 수립 및 기관 공식 계획 반영  
  - AI 기반 단속 효율화 성과를 정량 지표로 측정·보고하는 체계 구축  

- **데이터 공유·협력 강화**:  
  - 전국 본부 검사원 대상 '현장 채증 영상·이미지 데이터 수집' 채널 구축  
  - 타 공공기관(한국석유공사, 한국가스안전공사 등)과 AI 활용 사례 벤치마킹 네트워크 구축  

## 2. 과정 참여 필요성  
이동판매차량 단속 업무에서 검사원들이 채증 영상을 수동으로 검토하는 비효율성을 해결하기 위해 YOLOv11m 모델을 독학하고, 1,000여 장의 이미지를 촬영·레이블링해 학습 데이터셋을 구축했습니다. 시스템 설계와 운영 프로세스까지 기획했으나, 모델의 안정적 배포와 고도화를 위해 체계적인 전문교육이 필요합니다. 본 과정의 Python 심화, 웹앱, 에이전트, 업무자동화 커리큘럼은 기술적 공백을 정확히 채워 줄 것입니다. 수료 후에는 검사원 누구나 쉽게 활용할 수 있는 완성된 서비스로 발전시키고, 기존 시스템과의 통합 플랫폼을 구축해 석유 불법유통 근절에 기여하겠습니다. 또한 전국 본부 검사원들이 AI 도구를 자신 있게 활용할 수 있도록 거점 리더로서의 역할을 성실히 이행하겠습니다.
