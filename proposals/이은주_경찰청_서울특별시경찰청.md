# [경찰청 서울특별시경찰청] 이은주 - 자기소개서 및 과제기획서

**성명:** 이은주

**소속:** 경찰청 서울특별시경찰청

---

| 1. 지원동기 |
| --- |
| - 참여동기 및 기대효과 인공지능전환팀에서는 해양경찰청에 적합한 인공지능 서비스를 도입하기 위한 다양한 고민을 지속해왔습니다. 이를 위해, 해양경찰에 특화된 AI 서비스를 기획하기 위한 AX 기획역량이 필요하고, 또한 최신 트렌드를 파악하고 실무에 최적화된 AI 서비스 도입을 위해 본 과정의 기획 프로젝트를 추진하며 범용 모델의 한계를 넘는 '해상 치안 특화 AI 플랫폼' 기획과 이해도를 향상하기 위해 지원하게 되었습니다. |
| 2. AI·데이터분석 활용 경력 및 실적 |
| - AI·데이터 분석 프로젝트 경험('22. 06 ~ 10) 재난 관련 트위터 데이터를 활용한 LLM(Large Language Model) 모델 파인튜닝 프로젝트는 자연어 처리 기술의 발전과 함께 재난 상황에서의 정보 분석 및 대응 능력을 강화하기 위해 수행하였습니다. 이 프로젝트는 BERT, RoBERTa와 같은 트랜스포머 모델을 기반으로 한 파인튜닝 및 데이터 증강 기법을 활용하여 모델의 성능을 최적화하는 방법을 테스트했습니다. - AI·데이터 분석 관련 업무수행 경험 AI·데이터 분석 활용 경력 및 실적 초거대 AI 기반 플랫폼 이용지원 사업 ('24. 10. ~ 12): 해양경찰 특화 모델 생성 및 테스트를 담당하며, 공공 데이터 기반의 생성형 AI 활용 가능성을 검증함. |
| 3. 데이터 분석·AI활용 계획 |
| - 수행 프로젝트(확장예정)  AI 기술을 해양 재난 상황에 접목하여 사고 예측 및 대응 효율성을 높이는 대규모 R&D 프로젝트의 사업 관리를 맡아, 기술의 현장 적용 가능성을 면밀히 검토하고 있 습니다. <해양 치안 업무 보조 멀티모달 AI 에이전트 개발/미정> 해양경찰 내부 데이터와 범정부 공통기반 및 독자 파운데이션 모델을 활용한 업무 활용 에이전트 개발 계획이 있습니다. (1) 데이터 고도화: 산재된 법령 지침 및 사고 보고서를 파싱(Parsing)하고 청킹 (Chunking)하여 고성능 벡터 DB에 저장하는 RAG 파이프라인을 구축 (2) 행정 연동: 공문서 초안 작성, 법령 대조 등 실제 행정 절차를 수행하는 '에이전 트' 형태로 구현 |

# 2. 실전 AX 과제 기획서

| 1. 과제 개요 |
| --- |
| ○ 과제명: 해양경찰 행정 혁신을 위한 내부망 기반 멀티모달 RAG 및 지능형 에이전트 시스템 구축 ○ 추진배경 현재 해양경찰 업무는 고도의 전문성을 요하는 법령 해석과 방대한 현장 데이터 분 석이 병행되어야 합니다. 그러나 기존의 키워드 검색 방식은 맥락 파악에 한계가 있고, 신규 임용자나 현장 직원이 수천 페이지에 달하는 매뉴얼을 즉각 숙지하기에 는 물리적 시간이 부족하여 AX 전환이 시급한 상황 ○ 목표 내부 보안 가이드라인을 준수하면서도 최신 LLM 성능을 발휘하는 '해경 전용 지능 형 업무 비서'를 구축하여, 해양경찰 파출소 행정업무 5개 감축 목표 |
| 2. 현안 문제 및 데이터 현황 |
| ○ 문제정의 방대한 매뉴얼 지침과 단순 반복적인 파출소 행정업무로 인해 실시간 현장 대응 속 도가 저하되고, 업무 효율성이 감소하는 문제가 발생 ○ 보유 데이터 해상 치안 법령 및 내부 지침서(PDF) 방대안 200여 매뉴얼 및 파출소 현장업무일지 |
| 3. 구현 방안 |
| Hybrid RAG 구축: 문서 구조 분석(Layout Analysis) 모델을 적용하여 표와 그림이 포함된 복잡한 해경 문서를 완벽하게 벡터화하겠습니다. On-premise Fine-tuning: 해경 전문 용어(해상 약어, 관용 표현 등)를 학습시킨 소형 언어 모델(sLLM)을 내부 환경에 구축 MCP 기반 Tool 연동: AI가 내부 DB 조회, 공문서 서식 자동 생성 등 실제 업무 툴 을 제어할 수 있는 표준 프로토콜을 구현 |
| 4. 현장 이행계획 및 기대효과 |
| ○ 현장 이행계획 해양경찰 인공지능전환팀의 빅데이터플랫폼에 시범 운영 ○ 기대 효과 파출소의 단순업무 경감으로 직원 업무 만족도 제고 |
