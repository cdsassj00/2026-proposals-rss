# 1. 지원동기  
- **참여동기**  
  - 2022년부터 '인재개발플랫폼' 내 AI 및 빅데이터 분석 기능 구축·운영을 담당하며 전문성 강화 필요성을 체감  
    - 데이터 분석 및 AI 활용 역량 부족으로 자체 학습 병행했으나 한계 존재  
    - 2024년 역량강화 교육 수료 후 데이터 분석 전문성 중요성 인지, 2025년 빅데이터학과 석사 과정 진학 예정  
    - 이론 학습과 실무 적용 간 격차 해소를 위해 프로젝트 중심 교육 필요성 절감 → 본 교육 과정 지원 결정  

- **기대효과**  
  - 실무 적용 가능한 AX 기술 습득 및 기관 현안 해결을 위한 AI 전환 선도  
  - 생성형 AI 기반 데이터 분석 방법 이해, AI 에이전트 실습, 실전 프로젝트 수행을 통한 개발 역량 강화  
  - 데이터기반행정 거점으로서 행정-기술 간 소통 창구 역할 수행 및 리더 역량 함양  

- **관련 교육 경험**  
  - 데이터기반행정 역량강화 교육(2024년)  
  - 모델 파인튜닝·RAG 실습(2024년~)  
  - 빅데이터학과 석사 과정 진학 예정(2025년)  
  - AICE Associate 자격증 취득(2025년), ADsP 취득 및 석사 논문 준비 중(2026년)

# 2. 경력 및 실적  
## AI·데이터 분석 프로젝트 경험  
### ① 유사 직무 키워드 군집 자동화  
- **개요**: 인재개발플랫폼 내 직무 키워드 전처리 시간 단축을 위한 LLM 파인튜닝 및 FastText 임베딩 시도  
- **결과**: 초기 학습 부족으로 실패 → 향후 임베딩 방식 개선 필요  

### ② RAG 기반 데이터 전처리 방안 검토  
- **개요**: AI Hub 대학 강의 전사 데이터 활용해 불필요한 단어·문장 필터링 및 TF-IDF 기반 키워드 도출  
- **결과**: 전처리 후 검색 성능 유지 확인 → 데이터 품질 관리 방안 마련  

### ③ RAG 기반 반려견 건강 챗봇 구현  
- **개요**: AI Hub 말뭉치 데이터 활용해 파인튜닝 및 RAG 구성  
- **결과**: 일부 질의에 무의미한 출력 발생 → 데이터 전처리 및 분할 전략 보완 필요  

## AI·데이터 분석 업무 경험  
### ① 직무 키워드 기반 부서별 대표 직무 분석  
- **개요**: 인재개발플랫폼 구축 시 희망부서 맞춤형 추천 기능 구현을 위한 직무 키워드 군집화  
- **결과**: Word2Vec을 통한 유사 키워드 도출 및 추천 시스템 구현  

### ② BERT 기반 콘텐츠 부적합 자동 판별 모델 구축  
- **개요**: YouTube 콘텐츠 품질관리 자동화를 위한 BERT 모델 학습  
- **현황**: 전처리 전략 마련 후 테스트 데이터 분류 성능 개선 중

# 3. 실전 AX 과제 기획서  
## 1. 과제 개요  
- **과제명**: AI 기반 콘텐츠 모니터링 자동화 방안 실증  
- **추진배경**: 월평균 6.8만 건의 신규 콘텐츠 인입 대비 품질관리 인력·시간 부족  
- **목표**:  
  - 정량적: 자동화 후 성과 지표 설정 및 점검  
  - 정성적: 온라인 교육 환경 개선 및 이용자 만족도 향상  

## 2. 현안 문제 및 데이터 현황  
- **문제점**:  
  - 인력 한계로 콘텐츠 검토 범위 제한  
  - 담당자 주관 개입 가능성  
  - 수동 조사로 인한 업무 부담 가중  
- **보유 데이터**:  
  - 내부/외부 교육자원, 동영상 콘텐츠, 학습자 정보 등 총 13,104,390건  

## 3. 구현 방안  
### 시스템적 접근  
1. **학습 콘텐츠 분석 시스템 구축**  
   - 메타데이터 수집, 음성 데이터 추출(STT 모델), 텍스트 요약 모델 도입  
2. **모니터링 자동화 시스템 구축**  
   - RAG 데이터 구축, 외부 데이터 검색 기능, LLM 기반 모니터링 시스템 구현  
3. **사용자 기능 구현**  
   - 모니터링 API 및 대시보드 개발  

### 행정적 접근  
- 품질관리 절차 개편, 시범운영을 통한 유효성 검증, 학습자·관리자 의견 수렴  

## 4. 현장 이행계획 및 기대효과  
- **이행계획**:  
  - GPU 인프라에 AI 모델 설치, 기존 유지관리 사업과 연계해 운영  
  - 단기: 대화형 서비스 완성 → 중장기: 공공·민간 확산  
- **기대효과**:  
  - 연간 3,400시간, 1.2억 원 절감  
  - 생성형 AI 성능 향상, 챗봇 인터페이스 도입, API·모델 공개를 통한 확산

## 1. 데이터기반행정 거점 리더 활동 계획

- **기관**: 데이터 분석 문화 확산 및 기관 간 가교 역할 수행
