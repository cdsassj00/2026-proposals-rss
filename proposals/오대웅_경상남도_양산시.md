## 1. 지원동기

저는 2024년 2월, 양산시에서 인공지능 연구 동호회를 설립하여 활동 중입니다. 현재 대규모 언어 모델의 핵심 기능인 Structured model outputs와 Function calling 등을 연구하고 있으며, 연구 결과와 API 기반의 인공지능 모델을 활용하여 공공 업무 서비스를 개발하고 있습니다. 또한, 타시군 정보화 공무원을 대상으로 Python 강의와 같은 활동도 꾸준히 진행하고 있습니다. 하지만, 자체 연구만으로는 행정에 접목하는 데 한계를 느끼고 있습니다. 이에 정 부에서 제공하는 전문 교육 프로그램과 인증 자격을 통해 기관의 전문 인재로 성장 하여 지역 내 인공지능 역량을 높이고자 합니다.
## 2. AI·데이터분석 활용 경력 및 실적

( AI·데이터 분석 프로젝트 경험 ) ○ 2021년 인재를 유인하기 위한 빅데이터 분석 사업추진 · 양산시 대내 및 외환 위기상황에서 양산시의 인구분석을 시작으로 경제, 산업, 미래유망산업 선정 등 빅데이터 분석 추진 · '23년 산자부 미래차 재생배터리 공모사업 국비 86억 확보에 분석결과 활용 · 행정안전부 제39회 지역정보화 연구과제 발표대회 장관상 수상 ○ 2022년 방범용 CCTV 영상을 활용한 도로교통량 AI 분석 모델 개발 · Yolov5 모델을 이용해서, 방범용 카메라 속 차량 카운터 모델 개발 · 국토부에서는 매년 10월 셋째주 전국 동시 교통량 조사에 모델 적용 협업 ○ 2022년 인공지능 모델을 활용하여 축제 기간 이동식 유동인구 분석 모델 개발 · Yolov5 모델 활용 사람 탐지 및 통계 카운터 모델 개발 · 행안부 2022년 지방자치단체 빅데이터 분석 사례집 우수사례 소개 · 물금 벚꽃 행사를 인공지능 분석을 통해 "축제"로 승격 반영 ( AI·데이터 분석 관련 업무수행 경험 ) ○ 16종 차량 인식 학습 데이터 구축과 Python 기반 인공지능 모델 학습 경험 ○ 생성형 인공지능 모델의 API 서비스를 통해, 공공용 생성형 AI 서비스 자체 개발 ○ "양산시 인공지능 어시스턴스 클라우드 사이트"(https://m.site.nave )운영 중 해당 사이트는 온나라 자료실을 통해 전국 공무원에게 연구 결과 공개 ○ 행정업무용 자동화 RPA 프로그램 다수 개발 지원

# 2. 실전 AX 과제 기획서

1. 과제 개요

○ 과 제 명 : 인공지능 대전환(AX)을 위한 행정 맞춤형 통합 데이터베이스 구축 ○ 추진배경 · 지방정부의 중점 시책, 신규 사업, 국도비 신청 보고 등 기관에서 추진하는 중요한 문서들이 한글 파일로 작성되지만, 문서 취합 및 내용 검색이 불편하여 활용도가 낮음 정책의 일관성 확보 및 AX 대전환을 위해 인공지능이 활용 가능한 비정형 데이터베이스로 전환 필요 ○ 목표 (정량 목표) 2027년 민선 9기의 중점시책, 신규(특화)사업 통합 데이터베이스 구축 · 인공지능 활용을 위한 한글파일 확장자 .hwpx 전면 사용 및 문서 파싱 파이프라인 구축 · AI가 잘 이해하는 Json 구조의 비정형 MogoDB 데이터베이스 구축 · 오픈소스 모델 Gemma4:4B(Google사) 수준 인공지능 Rag 서비스 구축 (정성 목표) 민선 9기, 지방정부 인공지능 AX 전환 대전환 시작 · 행정 및 내부 데이터를 활용한 자치단체 특화된 인공지능 대전환(AX) 기반 조성 · 공공 업무의 효율성을 높이고 공공 Agent 지원 행정 문화 조성 · 지속 가능한 AX 혁신을 통해 시민들에게 안전한 인공지능 서비스 제공
2. 현안 문제 및 데이터 현황

○ 20년 정보화 추진 경험을 바탕으로 "딸깍 AI 모티브" 문화 확산 · 거점 전문 리더와 협업하여 "당신의 아이디어를 현실 구현"으로 업무 자동화 가능 · 부서별 AX 업무 협업으로 성과 창출 사례 발굴 ○ 보유한 서버를 활용한 로컬 LLM(Gemma3:4B) 분산 활용 추진 · 큰 컴퓨팅 파워가 필요한 고성능 모델 대신, 업무별 작은 모델 맞춤형 지원 단순 질의 응답용 LLM 서버, 이미지 생성 서버 등 유휴 서버 재정비 추진
### 2. 과정 참여 필요에 대해 자유롭게 기술

○ 2020년부터 인공지능과 빅데이터 관련 업무를 추진해왔으나, 내부 인사 방침 으로 인해 업무의 연속성이 끊어졌습니다. 이로 인해 당시의 개인적인 노력과 성과가 사라졌으며, 후임자 또한 전문성있는 사업으로 많은 어려움을 겪었습니다. ○ 업무와는 별개로 개인 시간을 활용하여 인공지능 연구를 꾸준히 진행한 덕분에 많은 성과를 이룰 수 있었습니다. 이 기술을 전문 인재 양성과정을 통해 공공 "자격"을 획득하고, 양산시의 지속 가능한 AI 전문 인재로서 인공지능 활용 역량 강화에 이바 지하고자 신청합니다.
