## 3. 실전 AX 과제 기획서

### 1. 과제 개요
- **과제명**: 「세종 스마트 일상감사」 — 물품·용역계약 통합 AI 일상감사 자동화 시스템 구축
- **추진 배경**:
  1. 1억 원 이상 사업 증가로 감사관실 업무 부담 가중
  2. 경북교육청 AI 일상감사 2.0 사례 벤치마킹
  3. 용역계약 영역까지 적용 범위 확장 (NLP·RAG 기술 활용)
- **목표**:
  - 정량적: 일상감사 문서 1차 자동 검토 80% 이상, 감사관 검토 시간 50% 단축
  - 정성적: 전국 두 번째 AI 일상감사 시스템 구축 + 용역계약 최초 적용 사례

### 2. 현안 문제 및 데이터 현황
- **문제 정의**:
  - 감사관실: 양적 과부하, 반복 업무 피로, 근거 추적 어려움
  - 피감 부서: 사전 체크리스트 부재, 반복 보완 요청
- **보유 데이터**:
  - 내부: 과거 일상감사 이력, 감사 규정 문서, 에듀파인 데이터
  - 외부: 국가법령정보센터, 조달청 나라장터, 공공데이터포털

### 3. 구현 방안
- **시스템 아키텍처**:
  1. **1단계 (물품구입)**: 규칙 기반 + ML 하이브리드 모델
  2. **2단계 (용역계약)**: BM25·임베딩 기반 유사도 분석, RAG 기반 규정 매칭
- **기술 스택**:
  - LLM: 오픈소스 sLLM (Polyglot-Ko, Solar-Ko 등)
  - 임베딩: KoSimCSE, BGE-M3-Ko + ChromaDB
  - 문서 전처리: hwp5/pyhwp, PyMuPDF
  - RAG: LangChain/LlamaIndex 기반 파이프라인
- **AI 조력자 원칙**: 감사관 판단 보조, 재량권 침해 방지

### 4. 현장 이행계획 및 기대효과
- **이행계획**:
  1. 1단계 (2026.5.~6.): 경북 모델 벤치마킹, TF 구성
  2. 2단계 (2026.7.~9.): 물품구입 PoC 구축
  3. 3단계 (2026.10.~12.): 용역계약 모듈 확장
  4. 4단계 (2027.1.~6.): 본격 운영 및 확산
- **기대 효과**:
  - 예산 절감, 업무 시간 단축, 감사 품질 향상, 대국민 서비스 개선
