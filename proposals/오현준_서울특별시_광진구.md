# 의성군 AI·데이터분석 전문인재 양성과정 지원서

## 1. 지원동기

### 참여동기 및 기대효과
행정안전부 재직 시절 「데이터기반행정 역량강화 사업」을 수행하며 공공기관의 AI·데이터 활용 수준을 진단하고 교육·컨설팅을 통해 현장 적용을 지원한 경험이 있습니다. 이 과정에서 AI 이해 및 활용 단계는 달성했으나, 실제 업무·정책 과정에 연계하는 데 한계가 있음을 확인했습니다.  
현재 의성군에서는 생성형 AI 기반 행정지원 플랫폼 구축, 대민 챗봇 서비스 도입, AI 활용 과제 발굴, 인재 양성체계 마련 등을 추진 중입니다. 단순 데이터 분석을 넘어 AI를 행정업무에 적용하는 기획 및 서비스 구체화 업무를 수행 중이며, 단순 기술 구현만으로는 업무 개선이 어렵다고 판단했습니다.  
행정 현장에 정착시키기 위해서는 부서별 업무 흐름에 맞는 적용 방식, 정책 검토·의사결정 연계 기준, 지속적 운영 체계가 필요합니다. 본 과정은 현재 추진 중인 AI 기반 행정서비스와 업무 혁신 과제를 현장에 적용·정착시키는 데 필요한 역량을 강화할 수 있는 기회입니다.  
과정 참여를 통해 AI 서비스 기획, 업무 적용 방식, 운영 체계 설계를 구체화하고, 이를 의성군 여건에 맞게 발전시켜 나가고자 합니다. 또한 행정 데이터 분석, AI 모델 설계, 서비스 구현을 통합적으로 수행하는 실전 역량을 강화하고, 생성형 AI 플랫폼 기능 개발·기획에 직접 활용할 계획입니다.

### 관련교육 경험
- **융합기술경영(MOT) 석사과정**: 기술과 정책을 연계한 문제 해결 관점 습득, 토픽모델링 기반 텍스트마이닝 분석을 통해 지방정부 정책 이슈 도출 경험  
- **행정안전부 AI 챔피언 블루 종합과정(45시간)**: 생성형 AI 활용, 데이터 분석, 머신러닝·딥러닝 등 공공업무 적용 기술 학습  
- **인공지능 응용개발 및 빅데이터 분석 교육**: 데이터 분석·시각화 교육 강의 경험을 바탕으로 실무 적용 및 공유 경험 보유  

## 2. 경력 및 실적

### AI·데이터 분석 프로젝트 경험
#### ① 농식품 빅데이터 플랫폼 구축 (2021.6 ~ 2021.12 / 6개월)
- ELK(Elasticsearch, Logstash, Kibana) 기반 빅데이터 관리 도구 개발  
- ERP·ESL 시스템 DB에서 거래·유통·상품 데이터 및 이벤트 로그 수집 파이프라인 구축  
- Elasticsearch 역인덱스 기반 전문 검색 및 BKD Tree 구조 수치·위치 데이터 저장 체계 설계  
- Logstash 전처리 파이프라인 및 Kibana 시각화 대시보드 개발로 실시간 모니터링 환경 구현  
- 컨테이너 기반 가상화 기술 적용 및 SSL 인증·X-Pack 보안 설정으로 데이터 보안 체계 확보  
- Python(eland, elasticsearch-dsl)과 Elasticsearch 연계 분석 환경 구축, 81만 건 이상 거래 데이터 실시간 처리  

#### ② 농산물 수급패턴 식별 인자 기반 학습데이터 구축 및 분석 모델 설계 (2021.1 ~ 2021.2 / 2개월)
- 코로나19로 인한 농산물 수급 패턴 변화 대응을 위한 이상치 탐지 모델 개발  
- 13,515명 개인 고객 및 4,423명 사업자 고객 데이터 분석, 20여 개 식별 변수 설계  
- 고객 소비 패턴을 Life Cycle 5단계 및 성향별 그룹으로 세분화하여 분석 체계 마련  
- 랜덤포레스트 기반 이상치 탐지모델 설계(정확도 82.71%, 재현율 88.92%)  
- ‘직전 6개월 대비 최근 3개월 농산물 수급금액 증감률’이 핵심 영향 변수로 확인  

#### ③ 생성형 AI 기반 데이터 분석 및 LLM 챗봇 개발 실증 프로젝트 (2025.11. / 1개월)
- WIPS ON 기술특허 데이터 활용 특허 Q&A 챗봇 개발 교육 진행  
- 대화 이력을 반영한 메모리 기반 챗봇 구조 구현(OpenAI API, Gradio UI)  
- RAG 구조 적용: PDF·XLSX·CSV 등 특허 데이터 전처리·임베딩 후 Chroma 벡터DB 저장  
- LangChain Retriever로 사용자 질의와 관련된 특허 정보 검색 후 프롬프트 결합  
- 환각 가능성 감소를 위한 검색 결과 중심 프롬프트 적용 및 Gradio·Cloud Run 배포  

## 3. 실전 AX 과제 기획서

### 1. 과제 개요
- **과제명**: 의성군 국·도비 보조사업 발굴·신청 지원 AI Agent 구축(PoC)  
- **추진배경**: 인구감소·초고령화로 재원 확보 필요성 증대, 국·도비 보조사업이 핵심 재원  
- **기본방향**:  
  - 다중 Agent 기반 업무혁신(공고 발굴, 부서 매칭, 적합성 판단, 신청서 초안 작성 등)  
  - 외부정보 연계 표준화(국가보조금통합관리시스템, 정부24, e나라도움 등)  
  - 행정 책임성 확보(AI는 지원, 최종 결정은 부서 수행)  
  - 확산기반 확보(모듈형 구조로 타 지자체 확산 가능)  

### 2. 현안 문제 및 데이터 현황
- **현안 문제**:  
  ① 분산된 공고채널로 상시 모니터링 필요  
  ② 부서 매칭 지연으로 신청 준비 시간 소요  
  ③ 정책기조 일관성 부족  
  ④ 조직지식 활용 미흡  
- **데이터 현황**:  
  - 내부: 표준사업계획서, 지원사업 분석자료, 디지털 혁신 전략 보고서 등  
  - 외부: 정부24, e나라도움, 국가보조금통합관리시스템 등 공개 API  

### 3. 구현 방안
- **구현 방향**:  
  - 공고 수집 → 후보사업 분류 → 부서 추천 → 적합성 검토 → 신청서 초안 작성 단계별 Agent 구현  
  - 외부 공고정보 통합 수집·정리, 내부 자료 검토 기준 활용  
- **서비스 구성**:  
  - **Agent 역할분담**: 공고 수집, 사업 분류, 부서 매칭, 적합성 검토, 초안 작성  
  - **외부 연계**: 국가보조금통합관리시스템, 정부24, e나라도움 등 다중 채널 모니터링  
  - **검증 방안**: 3~5개 과제 선정 후 정확도·적정성·합리성·활용도·시간 절감 효과 검증  

### 4. 현장 이행계획 및 기대효과
- **이행계획**:  
  1단계(외부 정보 연계) → 2단계(내부 자료 정리) → 3단계(Agent 구현) → 4단계(시범 적용) → 5단계(기능 보완) → 6단계(확산 기반 마련)  
- **기대효과**:  
  - 공모 대응체계 고도화(상시 점검 → 후보사업 선별 → 부서 검토 → 신청 준비 체계화)  
  - 군정 연계형 재원확보 및 사업관리 강화  
  - 의성군형 AI 행정모델 구현 및 확대 적용 기반 확보  

## 4. 데이터기반행정 거점 리더 활동 계획

### 데이터기반행정 거점 리더 활동 계획
- **실무 중심 확산**: 교육 과정에서 습득한 데이터 분석 방법·생성형 AI 활용 사례를 의성군 업무에 적용(보고서 작성 지원, 자료 요약, 반복업무 개선 등)  
- **현안 연계 과제 발굴**: 민원·주민 수요·재난 대응·행정 효율화 등 실제 현안과 연결된 데이터 활용 과제 발굴  
- **부서 간 데이터 연계**: 데이터센터 구축사업으로 축적된 데이터를 부서 간 공동 활용 체계로 확대, 외부 공공데이터와의 연계 검토  
- **성과 공유**: 적용 결과·한계를 내부 공유, 활용 사례 지속적 축적  

### 과정 참여 필요성
의성군은 생성형 AI 플랫폼 구축, 데이터 기반 정책 현안 발굴, AI 인재 양성 등을 추진 중이나, 시스템 구축만으로는 부족하며 실제 업무 적용이 더 중요합니다. 본 과정은 현재 과제를 행정 절차와 업무 흐름에 맞게 안착시키는 역량을 강화할 수 있는 기회입니다. 과정에서 습득한 내용은 개인 학습에 그치지 않고 내부 업무에 직접 적용하며, 조직 내 실질적 업무 방식으로 확산할 계획입니다.
