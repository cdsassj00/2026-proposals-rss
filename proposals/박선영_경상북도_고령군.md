# 1. 지원동기  
저는 2009년 소방 공채 임용 후 재난 현장에서의 오랜 경험을 바탕으로, 재난 대응 방식의 실질적 변화를 고민했습니다. 2022년 ChatGPT 등 AI 기술의 발전은 이러한 고민을 행동으로 전환하는 계기가 되었으며, 휴직 후 인공지능학 전문학사 과정을 이수하며 기술적 역량을 쌓았습니다. 현재 서울종합방재센터에서 'AI 기반 재난종합상황관리 시스템 구축' 사업 감독 업무를 수행하며, 학문과 현장 경험을 결합하고 있습니다.  

본 과정은 도메인 지식, 기술적 역량, 실무 감각을 통합해 실질적 결과물을 창출할 수 있는 최적의 기회로, **"AI 기반 적응형 119 상황요원 트레이너"** 개발을 통해 교육·훈련 시스템의 표준화와 지속 가능한 역량 강화를 목표로 합니다.

# 2. 경력 및 실적  
## 1) AI·데이터 분석 프로젝트 경험  
- **지능형 맞춤 일정 추천 웹 플랫폼 'SmartDay' 구축** (캡스톤 프로젝트)  
  - **역할**: AI 파트 전담, RAG 및 Function Calling 기법 적용, LLM API 연동  
  - **성과**: FastAPI 백엔드 서버 구현, MySQL 데이터베이스 관리, Docker 기반 클라우드 배포 경험 확보  

## 2) AI·데이터 분석 관련 업무 경험  
- **'AI 기반 재난종합상황관리 시스템 구축' 사업 감독**  
  - **활동**: 외부 업체의 '상황보고서 자동 생성' 및 '구급상황관리 AI 모델' 현장 적합성 검토  
  - **성과**: LLM 생태계(RAG, LangChain 등) 이해를 바탕으로 기술적 방향 조율 및 실무자 의견 수렴

# 3. 실전 AX 과제 기획서  
## 1. 과제 개요  
- **과제명**: AI 기반 적응형 119 상황요원 트레이너 (『Adaptive 119 Dispatcher Trainer AI』)  
- **추진 배경**:  
  - 교육 성과 표준화 및 안전한 훈련 환경 필요성  
  - 적응형 훈련 시스템을 통한 개인별 역량 강화  
  - 지속 가능한 훈련 체계 구축  

- **목표**:  
  - **정량적**: 신규 요원 실무 투입 기간 30% 단축, 골든타임 내 핵심 정보 추출률 90% 달성  
  - **정성적**: 초개인화된 상황관리 역량 배양, 경험과 데이터 융합 훈련 체계 고도화  

## 2. 현안 문제 및 데이터 현황  
- **문제정의**:  
  - 훈련 결과의 정량화 및 맞춤형 피드백 부재  
  - 개인 맞춤형 훈련 시스템 미비  

- **핵심 해결 과제**:  
  - 적응형 상태 제어(Adaptive State Control)  
  - LLM 기반 정량 평가 엔진 구축  

- **보유 데이터**:  
  | 데이터 종류 | 규모 및 출처 | 활용 방안 |  
  |-------------|---------------|-----------|  
  | 비식별화된 신고 통화 녹취록 | 수만 건 (서울종합방재센터) | AI 페르소나 학습 데이터 |  
  | 재난 유형별 표준작전절차(SOP) | 소방청/재난본부 매뉴얼 | RAG 기반 평가 기준 수립 |  
  | 우수 상황요원 대응 패턴 | 1,000건 내외 (생성 예정) | LangGraph 상태 전이 로직 설계 |  

## 3. 구현 방안  
- **PTT(Push-to-Talk) 기반 음성 인터페이스**:  
  - Whisper-STT 모델로 실시간 음성→텍스트 변환  
- **LangGraph 기반 적응형 엔진**:  
  - 훈련생 발화에 따라 AI 신고자의 감정 상태(패닉 지수) 동적 조절  
- **RAG 기반 평가 시스템**:  
  - SOP 지식 베이스 구축, 실시간 대응 품질 분석  

## 4. 현장 이행계획 및 기대효과  
- **이행계획**:  
  1. **도입기**: AI 로직 검증 및 스트레스 테스트  
  2. **확산기**: 신임 요원 교육 정식 편입 및 역량 인증제 도입  
  3. **안착기**: 상시 훈련 체계 정착 및 광역 확산  

- **기대효과**:  
  - 재난 대응 효율 극대화 및 대국민 서비스 혁신  
  - 교육 예산 절감 및 행정 효율성 제고  
  - 데이터 기반 맞춤형 역량 관리 체계 확립

# 4. 데이터기반행정 거점 리더 활동 계획  
## 1. AI·데이터 분석 활용 문화 확산  
- 사내 스터디 운영, 관리자 대상 AI 리터러시 교육, 실무자 대상 바이브 코딩 교육 제공  

## 2. 데이터 기반 정책 수립 및 제도 마련  
- 훈련 데이터 수집·활용 기준 수립, 데이터 거버넌스 표준 절차 개발  

## 3. 외부 데이터 연계 및 API·데이터 플랫폼 활용  
- RAG 구조 적용 SOP 매뉴얼-외부 데이터 실시간 참조 파이프라인 구축  

## 4. 데이터 공유·협력 강화  
- 행안부·NIA AI 거점 리더 네트워크 참여, 유관 기관과의 데이터 공유 협력 체계 구축  

## 2. 과정 참여 필요성  
16년의 소방 실무 경험, 인공지능학 전공 역량, AI 시스템 구축 사업 감독 경험을 바탕으로, 본 과정을 통해 **"Adaptive 119 Dispatcher Trainer AI"**를 완성하고 공공 부문 AX 전환의 실질적 사례를 만들겠습니다. 문제는 현장에 있으며, 저는 항상 그 현장에 있었습니다.
